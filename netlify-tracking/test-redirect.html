🔗 トラッキングリンク デバッグツール

📝 このツールの使い方：
  1. 作成したトラッキングコードを入力
  2. 「テスト実行」ボタンをクリック
  3. 結果とログを確認
  4. 問題がある場合は、Netlify Function ログも確認してください

🧪 テスト 1: トラッキングコードのテスト

📊 テスト 2: Function ログ確認

Netlify Function のログを確認する方法：

# Netlify Dashboard で確認
1. https://app.netlify.com にログイン
2. サイトを選択
3. Functions → track-redirect を選択
4. ログタブを確認

# または Netlify CLI を使用
$ netlify functions:log track-redirect

🔍 テスト 3: データベース確認

📜 実行ログ
