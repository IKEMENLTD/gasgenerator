====================================
Netlify環境変数設定（コピペ用）

====================================

【必須】Supabase設定
------------------------------------
Key: SUPABASE_URL
Value: （あなたのSupabase ProjectのURL）
例: https://abcdefghijklmn.supabase.co

Key: SUPABASE_ANON_KEY
Value: （あなたのSupabase Anon Key）
例: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

Key: SUPABASE_SERVICE_ROLE_KEY
Value: （あなたのSupabase Service Role Key）
例: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

取得方法: Supabase Dashboard → Settings → API


【必須】JWT認証
------------------------------------
Key: JWT_SECRET
Value: a8f3d2e9b1c4f7a6d5e8b2c9f3a1d4e7b9c2f5a8d1e4b7c9f2a5d8e1b4c7f9a2

（またはランダムな32文字以上の文字列）


【必須】LINE公式アカウント
------------------------------------
Key: LINE_CHANNEL_ACCESS_TOKEN
Value: （あなたのLINE Channel Access Token）

Key: LINE_CHANNEL_SECRET
Value: （あなたのLINE Channel Secret）

Key: LINE_OFFICIAL_URL
Value: https://line.me/R/ti/p/@your-line-id

取得方法: LINE Developers Console → Messaging API


【必須】管理者ログイン
------------------------------------
Key: ADMIN_USERNAME
Value: admin

Key: ADMIN_PASSWORD

（本番環境では必ず変更してください）


【オプション】アプリケーション設定
------------------------------------
