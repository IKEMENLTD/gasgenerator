=====================================
  研修受講履歴 使い方説明書
=====================================

■ このシステムでできること
  - 社内研修の一元管理（研修マスタ・スケジュール・社員情報）
  - 受講履歴の記録と進捗管理
  - 研修申込と上長・人事の承認ワークフロー
  - 必須研修の対象者管理と期限チェック
  - 受講後フィードバック・効果測定の収集
  - 有効期限管理と再受講アラート
  - 部署別・社員別・研修別のレポート集計
  - 研修費用の管理と分析
  - 自動メール通知（リマインド・承認・フィードバック依頼など）
  - 承認完了時のGoogleカレンダー自動登録

■ アクセス方法
  ブラウザでシステムのURLを開くとダッシュボードが表示されます。
  左側のサイドバーから各ページに移動できます。
  スマートフォンでは左上のメニューボタンからサイドバーを開きます。

=====================================
  各ページの説明
=====================================

------------------------------------
1. ダッシュボード
------------------------------------
  システムを開くと最初に表示される概要画面です。

  【統計カード】（画面上部）
    - 有効研修数: 現在有効な研修の総数
    - 今年度受講完了: 今年度（4月〜翌3月）の受講完了件数
    - 必須研修未完了: 必須研修が未完了の件数（黄色背景で警告表示）
    - 期限超過: 受講期限を超過している件数（赤背景で警告表示）
    - 今年度研修費用: 今年度の研修費用合計

  【グラフ】
    - 部署別受講率: 部署ごとの今年度受講率を横棒グラフで表示
    - カテゴリ別受講時間: 研修カテゴリごとの受講時間をドーナツグラフで表示
    - 受講者数推移（月別）: 過去12ヶ月の月別受講完了数を折れ線グラフで表示

  【一覧テーブル】
    - 今後の研修スケジュール: 直近の開催予定（最大10件）
    - 承認待ち: 申込ステータスが「申込中」のもの
    - 未受講者アラート: 対応が必要なアラート（最大20件）
    - 有効期限切れ接近: 有効期限が60日以内のもの（最大10件）
    - 満足度ランキング: 研修の平均満足度上位5件
    - 最近の受講完了: 今月の受講完了記録（最大10件）

------------------------------------
2. 研修マスタ
------------------------------------
  研修の基本情報を管理します。

  【一覧画面】
    - カテゴリ・ステータスでフィルタ、キーワード検索が可能
    - 「編集」ボタンで既存データを修正、「削除」ボタンで削除

  【新規登録 / 編集】
    「+ 新規登録」ボタンまたは「編集」ボタンでフォームが開きます。
    - 研修名（必須）
    - 研修カテゴリ: コンプライアンス / ビジネススキル / 技術 /
      マネジメント / 新入社員 / 安全衛生 / その他
    - 研修種別: 必須 / 選択 / 推奨
    - 対象者: 全社員 / 新入社員 / 管理職 / 特定部署 / 特定職種
    - 対象等級
    - 実施形式: 集合研修 / オンライン / eラーニング / OJT / 外部研修
    - 提供元
    - 研修時間（時間）
    - 受講期限: テキスト入力（例: 入社後90日、毎年12月）
    - 有効期限（年）: 数値で入力（例: 3 → 受講完了から3年有効）
    - 再受講要否: あり / なし
    - テスト有無: あり / なし
    - 合格基準点: テスト有りの場合に設定
    - 費用（円/人）
    - 定員
    - 研修資料URL
    - ステータス: 有効 / 無効 / 終了

------------------------------------
3. 研修スケジュール
------------------------------------
  研修の開催予定を管理します。

  【一覧画面】
    - ステータスでフィルタ、キーワード検索が可能
    - 研修名、開催日、時間、場所、講師、定員、申込数を表示

  【新規登録 / 編集】
    - 研修（必須）: 研修マスタから選択（選択すると定員が自動入力）
    - 開催日（必須）
    - 開始時間 / 終了時間
    - 開催場所/URL
    - 講師名
    - 定員
    - 申込締切日
    - 担当者: 社員マスタから選択
    - ステータス: 募集中 / 締切 / 実施済 / 中止

------------------------------------
4. 社員マスタ
------------------------------------
  社員の基本情報を管理します。

  【一覧画面】
    - ステータスでフィルタ、キーワード検索が可能

  【新規登録 / 編集】
    - 氏名（必須）
    - 氏名（カナ）
    - 部署ID / 部署名（必須）
    - チーム名
    - 役職
    - 等級/グレード
    - 入社日
    - 雇用形態: 正社員 / 契約社員 / パート / 派遣
    - 上長ID / 上長名
    - メールアドレス: 各種通知メールの送信先
    - ステータス: 在籍 / 休職 / 退職

  ※ 新規社員登録時、「必須」かつ「全社員」「新入社員」対象の研修が
    自動的に必須研修対象者管理に追加されます。

------------------------------------
5. 受講履歴
------------------------------------
  社員の研修受講記録を管理します。

  【一覧画面】
    - ステータスでフィルタ、キーワード検索が可能
    - ID、社員名、研修名、受講日、完了日、点数、結果、ステータスを表示

  【受講記録登録 / 編集】
    - 社員（必須）: 社員マスタから選択
    - 研修（必須）: 研修マスタから選択（選択すると時間・費用・形式が自動入力）
    - 開催: スケジュールから選択（任意）
    - 受講日 / 受講完了日
    - 受講時間（時間）
    - 受講形式
    - テスト点数: 入力すると合格基準点と比較して合格/不合格を自動判定
    - 受講ステータス: 申込済 / 受講中 / 完了 / 不合格 / キャンセル
    - 費用（円）
    - 修了証URL
    - 備考

  【自動処理】
    - ステータスを「完了」にすると:
      ・必須研修対象者管理のステータスが「完了」に更新
      ・有効期限がある研修は有効期限管理に登録/更新
      ・社員のメールアドレスにフィードバック依頼メールを自動送信
    - テスト結果が「不合格」の場合:
      ・社員のメールアドレスに再受験案内メールを自動送信

------------------------------------
6. 研修申込
------------------------------------
  研修への申込と承認ワークフローを管理します。

  【一覧画面】
    - ステータスでフィルタ、キーワード検索が可能
    - 上長承認・人事承認の状況を表示
    - 承認待ちの場合、承認/却下ボタンが表示される

  【新規申込】
    - 社員（必須）: 社員マスタから選択
    - 開催スケジュール（必須）: 「募集中」のスケジュールから選択
      （選択すると研修名・開催日・場所・定員が表示されます）

  【承認ワークフロー】
    1. 申込登録 → 上長にメール通知
    2. 上長が「上長承認」ボタンをクリック → 上長承認完了
    3. 人事が「人事承認」ボタンをクリック → 両方承認済みになると:
       ・申込者にメールで承認完了通知
       ・Googleカレンダーに研修予定を自動登録
         （開始/終了時間がある場合は時間指定、なければ終日イベント）
    ※ 上長・人事どちらの段階でも「却下」が可能

------------------------------------
7. 必須研修管理
------------------------------------
  必須研修の対象者と受講状況を管理します。

  【一覧画面】
    - ステータスでフィルタ、キーワード検索が可能
    - 社員名、部署、研修名、受講期限、残り日数、ステータスを表示
    - 残り日数は色分け表示（緑: 余裕あり / 橙: 14日以内 / 赤: 超過）

  【対象者追加】
    - 研修（必須）: 「必須」種別の研修のみ表示
    - 社員（必須）: 在籍社員から選択
    - 受講期限

  ※ 新入社員登録時に自動追加されるため、通常は手動追加は不要です。
  ※ 受講履歴で「完了」にすると自動的にステータスが更新されます。

------------------------------------
8. フィードバック
------------------------------------
  研修受講後のアンケート回答を管理します。

  【一覧画面】
    - キーワード検索が可能
    - 満足度・理解度・活用度・講師評価を一覧表示
    - 「詳細」ボタンで全項目を確認

  【フィードバック登録】
    - 社員（必須）: 社員マスタから選択
    - 研修（必須）: 研修マスタから選択
      （社員と研修を両方選ぶと受講履歴から開催ID・受講日を自動入力）
    - 満足度（5段階）（必須）
    - 内容の理解度（5段階）
    - 業務への活用度（5段階）
    - 講師評価（5段階）
    - 良かった点（自由記述）
    - 改善要望（自由記述）
    - 今後受けたい研修

  ※ 受講完了時にフィードバック依頼メールが自動送信されます。

------------------------------------
9. 効果測定
------------------------------------
  研修受講後の効果を定期的に測定・記録します。

  【一覧画面】
    - キーワード検索が可能
    - タイミング、知識定着度、行動変容度、上長評価、フォロー要否を表示
    - 「詳細」ボタンで全項目を確認

  【効果測定登録】
    - 社員（必須）: 社員マスタから選択
    - 研修（必須）: 研修マスタから選択
      （社員と研修を両方選ぶと受講日を自動入力）
    - 受講日
    - 測定タイミング: 受講直後 / 1ヶ月後 / 3ヶ月後
    - 測定日
    - 知識定着度（5段階）
    - 行動変容度（5段階）
    - 上長評価
    - 追加フォロー要否: 不要 / 要
    - 業務成果への影響（自由記述）
    - 具体的な活用事例（自由記述）

  ※ 受講完了から1ヶ月後・3ヶ月後に効果測定依頼メールが自動送信されます。
    （既に該当タイミングの測定が登録済みの場合は送信されません）

------------------------------------
10. レポート
------------------------------------
  各種集計データを6つのサブタブで確認できます。
  初回表示時にデータを一括読み込みし、タブ切替はキャッシュから表示します。

  【社員別】
    社員ごとの受講サマリーを表示。キーワード検索が可能。
    - 累計受講数・累計受講時間
    - 今年度受講数
    - 必須研修完了率（%）
    - 平均テスト点数・平均満足度
    - 最終受講日

  【部署別】
    部署ごとの月次受講サマリーを表示。
    - 年月・部署名・所属人数
    - 受講者数・受講率（%）
    - 総受講時間
    - 必須研修完了率（%）
    - 研修費用合計・平均満足度

  【研修別】
    研修ごとの年度実績サマリーを表示。
    - 開催回数・総受講者数
    - 受講率（%）
    - 平均テスト点数・合格率（%）
    - 平均満足度・総費用

  【アラート】
    日次チェックで検出されたアラートの一覧。
    - 検出日・アラート種別（期限接近/期限超過/再受講期限）
    - 社員名・研修名・受講期限
    - 残日数・超過日数・対応ステータス

  【有効期限】
    研修の有効期限管理データの一覧。
    - 社員名・部署・研修名
    - 受講完了日・有効期限・残り日数
    - 再受講要否・ステータス（有効/期限接近/期限切れ）

  【費用管理】
    月次の研修費用データの一覧。
    - 年月・研修名・カテゴリ・提供元
    - 開催回数・受講者数・単価・総費用

  ※ 社員別は週次、部署別・研修別・費用管理は月次で自動集計されます。

------------------------------------
11. アラート設定
------------------------------------
  自動アラートの条件を設定します。

  【一覧画面】
    - アラート名、条件、日数、対象、通知先、通知方法、状態を表示
    - 「編集」「削除」ボタンで管理

  【アラート追加 / 編集】
    - アラート名（必須）
    - 条件: 受講期限 / 有効期限 / 未申込 / テスト不合格
    - 条件値（日数）: アラートを発生させる残日数
    - 対象: 必須研修 / 全研修 / 特定研修
    - 通知先: 本人 / 上長 / 人事 / 本人+上長
    - 通知方法: メール
    - 有効/無効: アラートの有効・無効を切替

=====================================
  自動処理について
=====================================

以下の処理が自動で実行されます。手動操作は不要です。

  【毎日（日次チェック）】
    - 必須研修の受講期限チェック
      → 期限14日以内: 本人にリマインドメール
      → 期限超過: 本人にリマインド + 上長にエスカレーションメール
    - 有効期限チェック
      → 期限30日以内: アラート登録、ステータス更新
    - 研修前日リマインド
      → 翌日に研修がある承認済の受講者にメール通知
    - 効果測定依頼
      → 受講完了から1ヶ月後・3ヶ月後に測定依頼メール

  【毎週月曜（週次集計）】
    - 社員別受講サマリーの更新

  【毎月1日（月次集計）】
    - 社員別受講サマリーの更新
    - 部署別受講サマリーの更新
    - 研修別実績サマリーの更新
    - 研修費用管理の更新

=====================================
  共通操作
=====================================

  【検索】
    各ページの検索ボックスにキーワードを入力すると
    リアルタイムで一覧が絞り込まれます。

  【フィルタ】
    プルダウンでカテゴリやステータスを選択すると
    該当するデータのみ表示されます。

  【ページ送り】
    データが15件を超える場合、一覧の下部に
    「前へ」「次へ」ボタンと件数情報が表示されます。

  【フォーム入力】
    各ページの「+ 新規登録」ボタンでフォームが開きます。
    入力後「保存」ボタンで登録、「キャンセル」で閉じます。
    ※ 赤い * マークの項目は必須です。

  【削除】
    「削除」ボタンをクリックすると確認ダイアログが表示されます。
    「OK」で削除が実行されます。

  【通知メッセージ】
    操作の結果は画面右上に緑（成功）または赤（エラー）の
    メッセージで表示されます。3秒後に自動で消えます。

=====================================
  スプレッドシートのシート構成
=====================================

  以下の15シートで構成されています。
  システムから自動的に作成・管理されるため、
  直接編集する必要はありません。

  1.  研修マスタ
  2.  研修スケジュール
  3.  社員マスタ
  4.  受講履歴データ
  5.  必須研修対象者管理
  6.  研修申込データ
  7.  研修フィードバック
  8.  研修効果測定
  9.  社員別受講サマリー（週次/月次で自動集計）
  10. 部署別受講サマリー（月次で自動集計）
  11. 研修別実績サマリー（月次で自動集計）
  12. 未受講者アラートリスト（日次で自動検出）
  13. 有効期限管理（受講完了時に自動登録）
  14. 研修費用管理（月次で自動集計）
  15. アラート設定

=====================================
  運用の流れ
=====================================

  【初期設定】
    1. 社員マスタに社員情報を登録
    2. 研修マスタに研修情報を登録
    3. アラート設定で通知条件を設定

  【日常運用】
    1. 研修スケジュールで開催予定を登録
    2. 社員が研修申込から申込を行う
    3. 上長が申込一覧から承認（→ 人事が承認 → カレンダー登録）
    4. 研修実施後、受講履歴に受講記録を登録（ステータス: 完了）
       → フィードバック依頼メールが自動送信
    5. 社員がフィードバックを回答
    6. 1ヶ月後・3ヶ月後に効果測定依頼メールが届く
    7. 社員または上長が効果測定を回答

  【定期確認】
    - ダッシュボードで全体状況を確認
    - レポートで部署別・研修別の実績を分析
    - アラート一覧で未対応事項を確認
