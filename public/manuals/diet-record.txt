====================================
ダイエット記録 - 使い方説明書
====================================

------------------------------------
1. このシステムについて
------------------------------------

「ダイエット記録」は、体重管理・食事管理・運動管理を一元的に行うためのウェブアプリです。
日々の体重・体脂肪率の記録、食事内容と栄養素の記録、運動内容と消費カロリーの記録を
簡単に行うことができます。データはグラフやレポートで可視化され、目標達成に向けた
モチベーション維持をサポートします。

こんなことに使えます:
- 毎日の体重・体脂肪率を記録し、推移をグラフで確認する
- 食事の内容とカロリー・栄養素を記録し、食生活を振り返る
- 運動の種類・時間・消費カロリーを記録し、活動量を管理する
- 目標体重を設定し、進捗状況をリアルタイムで把握する
- 週次・月次のレポートで長期的な傾向を分析する
- バッジ・アチーブメント機能で継続のモチベーションを維持する
- 記録データをCSV/JSON形式でエクスポートする

------------------------------------
2. ログイン・新規登録
------------------------------------

[新規登録]
1. アプリを開くとログイン画面が表示されます
2. 「新規登録」リンクをクリックします
3. 以下の情報を入力します
   - ニックネーム（1〜20文字）
   - メールアドレス
   - パスワード（6文字以上）
   - 性別
   - 生年月日
   - 身長（cm）
   - 現在の体重（kg）
   - 目標体重（kg）※現在の体重より少ない値
   - 目標達成期日
   - 目標摂取カロリー（kcal）
   - 目標消費カロリー（kcal）
4. 「登録する」ボタンをクリックします
5. 登録が完了するとダッシュボードが表示されます

[ログイン]
1. メールアドレスとパスワードを入力します
2. 「ログイン」ボタンをクリックします

------------------------------------
3. ダッシュボード
------------------------------------

ダッシュボードでは以下の情報を一覧で確認できます。

[統計カード]
- 現在の体重とBMI
- 目標体重と残りの減量kg数
- 目標達成までの残り日数
- 直近7日間の平均体重
- 今日の摂取カロリー（目標値との比較）
- 今日の消費カロリー（目標値との比較）

[目標達成進捗バー]
- 開始時体重から目標体重までの進捗率がパーセントで表示されます

[体重推移グラフ]
- 体重と体脂肪率の推移を折れ線グラフで表示します
- 目標体重のラインも重ねて表示されます
- 期間は「1週間」「1ヶ月」「3ヶ月」「全期間」から選択できます
- グラフ上のデータポイントにマウスを合わせると正確な数値が表示されます

[カロリー推移グラフ]
- 摂取カロリーと消費カロリーを棒グラフで比較表示します

[クイックアクション]
- 「体重を記録」「食事を記録」「運動を記録」「目標設定」のショートカットボタンから
  各記録画面に素早くアクセスできます

------------------------------------
4. 体重・体脂肪率記録
------------------------------------

1. ナビゲーションメニューから「体重記録」を選択します
2. 以下の情報を入力します
   - 記録日付（デフォルトは今日の日付）
   - 記録時刻（デフォルトは現在時刻）
   - 体重（kg）※20.0〜200.0の範囲、小数点以下1桁
   - 体脂肪率（%）※5.0〜60.0の範囲、小数点以下1桁
   - メモ（任意、200文字以内）
3. 「記録する」ボタンをクリックします
4. BMIが自動計算されます
5. 記録一覧が画面下部に表示されます
6. 各記録の「詳細」ボタンで詳細情報を確認できます
7. 「削除」ボタンで記録を削除できます（確認ダイアログが表示されます）

------------------------------------
5. 食事記録
------------------------------------

1. ナビゲーションメニューから「食事記録」を選択します
2. 以下の情報を入力します
   - 記録日付
   - 記録時刻
   - 食事種類（朝食/昼食/夕食/間食/その他）
   - 食事カテゴリ（和食/洋食/中華など）
   - 食事内容（例：鶏むね肉のサラダ）※入力中に過去の記録から候補が表示されます
   - カロリー（kcal）
   - たんぱく質（g）
   - 脂質（g）
   - 炭水化物（g）
   - メモ（任意、500文字以内）
3. 「記録する」ボタンをクリックします
4. 記録一覧が画面下部に表示されます

------------------------------------
6. 運動記録
------------------------------------

1. ナビゲーションメニューから「運動記録」を選択します
2. 以下の情報を入力します
   - 記録日付
   - 運動開始時刻
   - 運動カテゴリ（ウォーキング/ジョギング/筋トレ/ヨガ/水泳など）
   - 運動時間（分）※1〜360分
   - 推定消費カロリー（kcal）※カテゴリと時間を選ぶと自動計算されます
   - 運動詳細（任意、500文字以内）
   - メモ（任意、500文字以内）
3. 消費カロリーはMETs値と体重から自動計算されますが、手動で修正も可能です
4. 「記録する」ボタンをクリックします

------------------------------------
7. 記録履歴
------------------------------------

1. ナビゲーションメニューから「履歴」を選択します
2. 上部のフィルタボタンで表示対象を絞り込めます
   - 「全て」：体重・食事・運動の全記録
   - 「体重」：体重記録のみ
   - 「食事」：食事記録のみ
   - 「運動」：運動記録のみ
3. 記録はテーブル形式で表示され、日付の新しい順に並びます
4. データが多い場合はページネーションで切り替えられます
5. 各記録の「削除」ボタンから記録を削除できます

------------------------------------
8. レポート
------------------------------------

1. ナビゲーションメニューから「レポート」を選択します
2. 「週次レポート」「月次レポート」をタブで切り替えられます

[週次レポート]
- 各週の平均体重、体重変化、記録日数が表示されます
- システムからのアドバイスが表示されます
- 毎週月曜日に自動集計されます

[月次レポート]
- 各月の平均体重、体重変化、記録日数が表示されます
- システムからのアドバイスが表示されます
- 毎月1日に自動集計されます

------------------------------------
9. バッジ・アチーブメント
------------------------------------

1. ナビゲーションメニューから「バッジ」を選択します
2. 獲得可能なバッジの一覧が表示されます
3. 獲得済みのバッジは色付きで表示され、獲得日が記載されます
4. 未獲得のバッジは半透明で表示されます

[獲得可能なバッジ]
- 7日間連続記録：7日間連続で体重を記録
- 30日間連続記録：30日間連続で体重を記録
- -1kg達成：開始時から1kg減量に成功
- -3kg達成：開始時から3kg減量に成功
- -5kg達成：開始時から5kg減量に成功
- 目標体重達成：目標体重に到達
- 運動5回達成：運動を5回記録
- 運動20回達成：運動を20回記録

------------------------------------
10. 目標設定
------------------------------------

1. サイドメニュー（モバイル）またはダッシュボードのクイックアクションから
   「目標設定」を選択します
2. 以下の情報を入力します
   - 目標体重（kg）
   - 目標達成期日
   - 目標詳細（例：夏までに5kg減）
   - ご褒美（例：新しい洋服を買う、100文字以内）
   - 目標達成宣言文（100文字以内）
3. 「目標を設定する」ボタンをクリックします
4. 設定した目標はダッシュボードの進捗表示に反映されます
5. 過去の目標履歴も画面下部で確認できます

------------------------------------
11. プロフィール設定
------------------------------------

1. ナビゲーションメニューから「設定」を選択します
2. 以下の情報を編集できます
   - ニックネーム
   - 性別
   - 生年月日
   - 身長
   - 目標体重
   - 目標達成期日
   - 目標摂取カロリー
   - 目標消費カロリー
3. 現在のBMIが自動計算されて表示されます
4. 「設定を更新する」ボタンで変更を保存します
5. 「ログアウト」ボタンでログアウトできます

------------------------------------
12. データエクスポート
------------------------------------

1. サイドメニュー（モバイル）から「データエクスポート」を選択します
2. エクスポート対象を選択します
   - 全て
   - 体重記録
   - 食事記録
   - 運動記録
3. ファイル形式を選択します
   - CSV
   - JSON
4. 「エクスポート」ボタンをクリックするとファイルがダウンロードされます

------------------------------------
13. 通知機能
------------------------------------

- ナビゲーションバー右上のベルアイコンをクリックすると通知一覧が表示されます
- 未読通知がある場合、ベルアイコンに赤いバッジが表示されます
- 通知をクリックすると既読になります
- 以下のタイミングで自動通知が生成されます
  - 記録リマインダー（毎日）
  - 目標達成間近・目標達成時
  - 3日間記録が途絶えた場合

------------------------------------
14. 自動集計機能
------------------------------------

以下の集計処理が自動で実行されます:
- 日次集計：毎日AM3:00に前日のデータを集計
- 週次集計：毎週月曜日AM4:00に前週のデータを集計
- 月次集計：毎月1日AM5:00に前月のデータを集計
- データ整合性チェック：毎週日曜日AM2:00
- 古いログデータ削除：毎月1日AM6:00（365日以上前のログ）

※自動集計を有効にするには、GASエディタでinitializeSheets関数とsetupTriggers関数を
  初回のみ手動実行する必要があります。

------------------------------------
15. モバイル対応
------------------------------------

- スマートフォンでアクセスすると、自動的にモバイル向けのレイアウトに切り替わります
- 画面右上のハンバーガーメニューからサイドバーが開きます
- サイドバーから全てのページにアクセスできます

------------------------------------
16. 初回セットアップ（管理者向け）
------------------------------------

初回利用時に以下の手順を実施してください:

1. GASエディタを開く（スクリプトID参照）
2. initializeSheets関数を実行
   → 全シートのヘッダー行とマスタデータが自動作成されます
3. setupTriggers関数を実行
   → 自動集計・通知用のトリガーが設定されます
4. ウェブアプリにアクセスして新規登録を行う
