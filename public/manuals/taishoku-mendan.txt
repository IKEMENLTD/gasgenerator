退職面談記録システム 使い方説明書
========================================

左のサイドバーから各画面に移動できます。
スマホの場合は左上の「≡」ボタンでサイドバーを開閉できます。


■ ダッシュボード
----------------------------------------

システムを開くと最初に表示される画面です。

上部の4枚のカードに今月の概況が表示されます。
  - 今月の退職者数（前月との比較つき）
  - 離職率
  - 平均満足度（5段階）
  - 面談実施率・NPS

各カードをクリックすると関連する画面に移動します。

中段にはグラフが6つ並びます。
  - 離職率推移（直近12ヶ月の折れ線グラフ）
  - 退職理由構成比（円グラフ）
  - 部署別離職率（横棒グラフ）
  - 満足度スコア推移（折れ線グラフ）
  - 在籍期間別退職分布（棒グラフ）
  - 年度目標 vs 実績（棒グラフ）

下段には2つのリストがあります。
  - 未実施面談リスト … 面談がまだの退職予定者が表示されます。
    残日数が3日以内の方は赤字で強調されます。
  - 改善アクション進捗 … アクションの完了率・件数が表示されます。


■ 退職者管理
----------------------------------------

退職が決まった方の情報を登録・管理する画面です。

【退職者を登録する】
  1. 右上の「退職者登録」ボタンを押す
  2. 氏名・性別・生年月日・入社日・退職日・部署名・雇用形態・退職種別を入力
     （「*」がついた項目は必須です）
  3. 「保存」を押す
  → 年齢・在籍期間は退職日と入社日から自動計算されます。

【退職者の詳細を見る】
  一覧の各行にある「詳細」ボタンを押すと、登録内容を確認できます。

【絞り込み・検索】
  一覧の上にある検索欄・プルダウンで絞り込めます。
  - 氏名検索 … 名前の一部を入力すると自動で絞り込まれます
  - 部署 … プルダウンから選択
  - 退職種別 … 自己都合／会社都合／定年／契約満了／その他
  - 面談状況 … 済／未


■ 面談記録
----------------------------------------

退職者との面談内容を記録する画面です。

【面談記録を登録する】
  1. 右上の「面談記録登録」ボタンを押す
  2. 基本情報を入力（退職者ID・退職者名・面談日・担当者名・面談形式）
  3. 退職理由を選択・入力
     → 理由カテゴリを選ぶと「過去の類似事例」が自動表示されます。
       参考にしながら記録できます。
  4. 満足度スコアを入力（各項目1〜5のボタンを押す）
     - 上長との関係 / 同僚との関係 / 仕事のやりがい
     - 給与 / 労働時間 / キャリア成長 / 総合満足度
  5. 転職先情報・NPS（0〜10）を入力
     NPSは0〜6が批判者（赤）、7〜8が中立者（黄）、9〜10が推奨者（緑）です。
  6. 面談者所見・改善提案を記入
  7. 「保存」を押す
  → 保存すると退職者マスタの面談状況が自動で「済」に更新されます。
  → 理由カテゴリに応じた「改善アクション提案」が表示されることがあります。
     チェックを入れて「選択したアクションを登録」を押すと、
     改善アクションに自動登録されます。


■ 退職分析
----------------------------------------

退職データを多角的に分析する画面です。
上部のタブで5つの切り口を切り替えられます。

  月次推移 … 月ごとの離職率の変動をグラフと表で確認
  部署別 … 部署ごとの離職率を比較
  理由別 … 退職理由の分布・推移を確認
  在籍期間別 … 短期離職か長期在籍後の離職かを分析
  属性別 … 年代・性別・職種・雇用形態ごとの傾向を確認
            （プルダウンで分析軸を切り替え）

※ 分析データは自動更新されます。手動で更新したい場合は
  管理画面から「分析更新」を実行してください。


■ 退職予兆
----------------------------------------

退職の兆候がある社員を記録・管理する画面です。
高リスクの検出がある場合、サイドバーにバッジ（数字）が表示されます。

【リスク検出を登録する】
  1. 「リスク検出登録」ボタンを押す
  2. 社員ID・社員名・部署・シグナル・リスクレベルを入力
  3. 「保存」を押す

【対応を記録する】
  1. 一覧の「対応」ボタンを押す
  2. 上長通知の有無、対応ステータス、対応内容を入力
  3. 「更新」を押す

【絞り込み】
  ステータス（未対応／対応中／解決／退職）や
  リスクレベル（高／中／低）で絞り込めます。


■ 改善アクション
----------------------------------------

面談から見えてきた課題に対する改善施策を管理する画面です。
期限超過のアクションがある場合、サイドバーにバッジが表示されます。

【アクションを登録する】
  1. 「アクション登録」ボタンを押す
  2. 課題内容・改善アクション・担当者・優先度・期限を入力
  3. 「保存」を押す

【アクションを更新する】
  一覧の「編集」ボタンから、ステータスの変更や効果・結果の記入ができます。
  ステータスを「完了」にすると完了日が自動記録されます。

【絞り込み】
  ステータス（未着手／進行中／完了／保留／中止）や
  優先度（高／中／低）で絞り込めます。


■ カテゴリ管理
----------------------------------------

退職理由の分類を管理する画面です。
面談記録で選択する理由カテゴリをここで追加できます。

【カテゴリを追加する】
  1. 「カテゴリ追加」ボタンを押す
  2. 大カテゴリ（待遇・仕事内容・人間関係・キャリアなど）を選択
  3. 中カテゴリ・小カテゴリ（具体的理由）を入力
  4. 改善可能性（高／中／低／不可）と対応部門を選択
  5. 「保存」を押す

一覧の「使用回数」列で、各カテゴリがどの程度使われているか確認できます。


■ 目標管理
----------------------------------------

離職率や満足度の目標値を設定し、実績と比較する画面です。

【目標を設定する】
  1. 「目標設定」ボタンを押す
  2. 年度・指標名（全社離職率／1年未満離職率／面談実施率／平均満足度）を選択
  3. 目標値を入力（業界平均・前年実績は参考値として任意入力）
  4. 「保存」を押す

上部のグラフで目標と実績の比較が視覚的に確認できます。
目標達成は緑、未達は赤で色分けされます。


■ 共通操作
----------------------------------------

【通知メッセージ】
  操作が成功すると画面右上に緑色のメッセージが表示されます。
  エラーの場合は赤色で表示されます。数秒で自動的に消えます。

【削除確認】
  データ削除時は確認ダイアログが表示されます。
  「OK」を押すまで削除は実行されません。

【モーダル（入力画面）の閉じ方】
  - 右上の「×」ボタン
  - 「キャンセル」ボタン
  - Escキー
  いずれの方法でも閉じられます。

【ページ送り】
  退職者一覧・面談記録・改善アクション・退職予兆の各画面では、
  1ページ20件ずつ表示されます。
  テーブル下部のページ番号ボタンで移動できます。
