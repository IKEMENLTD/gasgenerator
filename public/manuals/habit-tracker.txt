================================================================
 習慣トラッカーシステム 使い方説明書
================================================================

----------------------------------------------------------------
1. 概要 - どんなことに使えるか
----------------------------------------------------------------

習慣トラッカーシステムは、日々の習慣を記録・追跡し、
継続をサポートするウェブアプリケーションです。

- 毎日の習慣をワンクリックでチェックイン記録
- 連続達成日数（ストリーク）を自動で追跡・表示
- 週間ヒートマップで達成状況を色で可視化
- バッジ・実績システムでモチベーション維持
- チーム機能でメンバーと達成状況を共有・応援
- 週次・月次レポートを自動生成しメール送付
- 目標と習慣を紐付けて「何のためにやるか」を明確化
- リマインド通知で「やり忘れ」を防止

活用例:
- 個人の健康管理（運動・食事・睡眠の習慣化）
- 学習の継続（英語学習・読書・資格勉強）
- 仕事の生産性向上（朝活・日報・振り返り）
- チームでの習慣チャレンジ（社内の健康促進など）

----------------------------------------------------------------
2. 初期設定
----------------------------------------------------------------

2-1. アプリを開く
  ウェブアプリのURLにアクセスします。

2-2. シートの初期化
  画面右上の「初期設定」ボタンを押します。
  スプレッドシートに全20シートとマスタデータが自動作成されます。
  （カテゴリ・バッジ・システム設定の初期値が投入されます）

2-3. ユーザー登録
  「+ 新規登録」ボタンを押し、氏名・表示名・メールアドレスを
  入力して登録します。

2-4. ユーザー選択
  画面上部のプルダウンから登録したユーザーを選択します。
  以後、選択したユーザーとして操作します。

----------------------------------------------------------------
3. 各タブの機能
----------------------------------------------------------------

[ Dashboard タブ ]
  - 本日の達成率、完了/予定数、最長ストリーク、獲得バッジ数を表示
  - 本日の習慣一覧を表示（未完了が上、完了済みは薄く表示）
  - 習慣をクリックするとワンクリックでチェックイン
  - 直近5週間のヒートマップで達成パターンを可視化
    緑=80%以上、薄緑=50%以上、黄=スキップ、赤=失敗、灰=未記録
  - 各習慣のストリーク状況をバーチャートで表示
  - 全習慣完了時は祝福メッセージが表示されます
  - 22時以降に未完了があると警告バナーが表示されます

[ Check-in タブ ]
  - 本日の進捗バーと習慣一覧を表示
  - 各習慣の「Record」ボタンから詳細チェックインが可能
    完了/スキップ/失敗を選択
    実施量（目標量がある場合）を入力
    気分スコア（1-5）を記録
    一言コメントを記録
  - チェックイン時にバッジ条件を満たすと即座に獲得通知

[ Habits タブ ]
  - 登録済み習慣の一覧を表示
  - All / Active / Paused / Archive でフィルタ切替
  - 「+ 新規追加」から習慣を登録
    習慣名（必須）
    詳細説明
    カテゴリ選択（健康・運動/食事・栄養/学習・読書 等9種類）
    頻度（毎日/特定曜日/週N回/月N回）
    難易度（1-5）
    目標量と単位（例: 30分、10ページ）
    アイコン文字
  - 各習慣の「Pause」で一時停止（リマインド停止、ストリーク凍結）
  - 「Archive」でアーカイブ（一覧から非表示、記録は保持）
  - 一時停止中の習慣は「Resume」で再開可能

[ Team タブ ]
  - チーム一覧を表示
  - 「+ 作成」でチームを新規作成（チーム名・説明・公開設定）
    作成時に6桁の招待コードが自動発行されます
  - 「参加」で招待コードを入力してチームに参加
  - チーム名をクリックするとチーム詳細画面へ
    メンバーランキング（月間チェックイン数順）
    各メンバーの本日達成率・月間チェックイン数・最長ストリーク
    1位のメンバーには王冠マーク
    コメントフィード（応援・称賛・アドバイス）
    「+ Comment」から応援コメントを投稿

[ Reports タブ ]
  - Weekly / Monthly タブで切替
  - 週次レポート: 対象週・週間達成率・完璧達成日数・気分スコア平均
    AIコメント（達成率に応じた所感・改善提案）
  - 月次レポート: 対象月・月間達成率・チェックイン回数・気分スコア平均
    AIコメント（月次サマリー・翌月の改善提案）
  - レポートは毎週月曜9:00/毎月1日9:00に自動生成
  - メール受信設定がONのユーザーにはメールでも送付

[ Goals タブ ]
  - 目標一覧を表示（進行中/達成/期限切れ）
  - 「+ 新規」から目標を登録
    目標名（必須）・詳細説明・期限・優先度
  - 進行中の目標は「達成」ボタンで完了に変更
  - 期限まで7日以内の場合はカウントダウン表示
  - 期限切れの目標は自動でステータス更新

[ Badges タブ ]
  - 獲得済みバッジと未獲得バッジを一覧表示
  - バッジにはレアリティ（コモン/レア/エピック/レジェンダリー）
  - 獲得条件例:
    3日連続達成、7日連続達成、30日連続達成、100日連続達成
    通算50回/100回/500回チェックイン
    応援コメント10件以上投稿
    はじめの一歩（初回チェックイン）
  - バッジ獲得時にポイントが付与されます

[ Settings タブ ]
  - 表示名・メールアドレス・電話番号の変更
  - リマインド設定（方法・時刻・有効/無効）
  - 習慣公開設定（全公開/チームのみ/非公開）
  - 週次レポート・月次レポートの受信ON/OFF

----------------------------------------------------------------
4. 自動処理（バックグラウンド）
----------------------------------------------------------------

以下のトリガーが自動実行されます:

- 毎時: リマインド通知（設定時刻前後30分のユーザーに送信）
- 毎日23:00: ストリーク途切れ警告（3日以上のストリークが危険な場合）
- 毎日0:30: ストリーク自動計算・更新
- 毎日1:00: バッジ獲得チェック
- 毎日2:00: 日次ダッシュボード集計・チームランキング更新
- 毎日9:00: 目標期限アラート（7日以内の目標を通知）
- 毎週月曜9:00: 週次レポート生成・メール送付
- 毎月1日3:00: 習慣別・カテゴリ別パフォーマンス集計
- 毎月1日9:00: 月次レポート生成・メール送付

トリガーの設定は、GASスクリプトエディタで
setupTriggers() 関数を1回実行してください。

----------------------------------------------------------------
5. スプレッドシートのシート構成
----------------------------------------------------------------

マスタ系（7シート）:
  マスタ_ユーザー / マスタ_チーム / マスタ_習慣カテゴリ /
  マスタ_習慣 / マスタ_目標 / マスタ_バッジ・実績 / マスタ_通知テンプレート

トランザクション系（6シート）:
  トランザクション_チェックイン記録 / トランザクション_ストリーク記録 /
  トランザクション_バッジ獲得履歴 / トランザクション_応援・コメント /
  トランザクション_週次レポート / トランザクション_月次レポート

集計系（5シート）:
  集計_日次ダッシュボード用 / 集計_チーム別ランキング /
  集計_習慣別パフォーマンス分析 / 集計_カテゴリ別達成分析 /
  集計_チーム月次サマリー

設定・ログ系（2シート）:
  設定_システム設定 / ログ_操作履歴

----------------------------------------------------------------
6. Tips
----------------------------------------------------------------

- 習慣は3〜5個から始めるのがおすすめです。多すぎると挫折しやすくなります。
- ストリークが途切れても「再スタート」できます。完璧を目指さず継続を重視しましょう。
- チーム機能で仲間と共有すると、お互いの達成が刺激になり継続率が上がります。
- 週次レポートの「AIコメント」を参考に、習慣の見直しを行いましょう。
- 達成率が50%未満の習慣は、難易度を下げるか頻度を減らすことを検討してください。
