=====================================
  スキルマップ管理システム 使い方説明書
=====================================


はじめに
--------
このシステムは、社員のスキル・資格を一元管理し、
スキルギャップ分析・属人化リスク検出・育成計画の管理を行うツールです。

PC・スマホどちらからでも使えます。
左上のメニューボタンからページを切り替えてください。

全9画面、16シート構成。Soft UIデザインでスマホ対応。


----------------------------------------------------------------------
  ダッシュボード（トップページ）
----------------------------------------------------------------------

開くとまず表示される画面です。全体の状況がひと目でわかります。

  上部の数字カード（KPI）:
    在籍社員数 / スキル登録数 / 資格登録数 /
    スキルギャップ / アラート件数 /
    進行中の育成計画 / 完了した育成計画

  グラフ:
    カテゴリ別スキル分布 ... スキルカテゴリごとの登録割合
    部署別平均レベル   ... 各部署のスキル平均レベル（横棒グラフ）
    育成計画 進捗状況  ... 未着手/進行中/完了/中止の件数

  属人化リスクスキル:
    1名しか保有していないスキルを表示。
    早めに育成計画を立てることを推奨します。

  スキルギャップ TOP10:
    ギャップ（必須レベル - 現在レベル）が大きい順に表示。

  資格期限アラート:
    期限が近い/切れた資格を残り日数順に表示。
    赤字の残日数は早めに対応が必要です。

  テスト用データ管理カード:
    「仮データ投入」... テスト用の社員・スキル・資格データを一括投入
    「全データ初期化」... 全シートのデータを削除（ヘッダー行は残る）


----------------------------------------------------------------------
  スキル登録
----------------------------------------------------------------------

社員のスキルを自己評価で登録する画面です。

  登録するには:
    1. 社員を選択
    2. スキルカテゴリ → スキル項目を選択
    3. 現在レベル（1〜5）を選択
       レベルを選ぶとスキル定義が表示されます
    4. 必要に応じて目標レベル・経験年数・エビデンス・備考を入力
    5. 「登録」ボタンを押す

  登録済みスキル確認:
    下部の「登録済みスキル確認」で社員を選び「表示」を押すと、
    自己評価・上長評価・確定レベルの一覧を表示します。

  ポイント:
    - レベル1〜5はスキルごとに定義が異なります
    - 登録後、上長評価を受けると確定レベルが計算されます
    - 確定レベル = (自己評価 + 上長評価) / 2 の平均値


----------------------------------------------------------------------
  資格登録
----------------------------------------------------------------------

社員が取得した資格を登録する画面です。

  登録するには:
    1. 社員を選択
    2. 資格を選択
    3. 取得日を入力
    4. 証明書URLがあれば入力
    5. 「登録」ボタンを押す

  登録済み資格確認:
    下部の「保有資格確認」で社員を選び「表示」を押すと、
    取得済み資格の一覧（有効期限・ステータス含む）を表示します。

  ポイント:
    - 有効期限は資格マスタの「有効期限(年)」から自動計算
    - 期限が近づくとアラートリストに自動追加されます


----------------------------------------------------------------------
  上長評価
----------------------------------------------------------------------

上長が部下のスキルを評価する画面です。

  評価するには:
    1. 上長（自分）を選択
    2. 「未評価一覧を表示」を押す
    3. 部下が自己登録したスキルのうち、未評価のものが一覧表示
    4. 評価レベル（1〜5）を選択し「評価する」を押す

  確定レベルの計算:
    確定レベル = (自己評価レベル + 上長評価レベル) / 2
    小数点以下は四捨五入されます。

  ポイント:
    - 社員マスタの「上長ID」で上長と部下の紐付けを行います
    - 上長IDが設定されていないと未評価一覧に表示されません


----------------------------------------------------------------------
  育成計画
----------------------------------------------------------------------

スキルギャップを埋めるための育成計画を作成・管理する画面です。

  作成するには:
    1. 対象社員を選択
    2. 対象スキルを選択
    3. 現在レベル・目標レベルを入力
    4. 目標達成期限を設定
    5. 育成方法（OJT/研修/自己学習/メンタリング など）を入力
    6. 具体的アクション・担当者/メンターを入力
    7. 「作成」ボタンを押す

  進捗管理:
    一覧の「更新」ボタンでステータスを変更できます。
      未着手 → 進行中 → 完了 / 中止
    進捗メモも記録できます。
    「完了」にすると達成日が自動記録されます。

  フィルター:
    社員・ステータスで絞り込み表示が可能です。


----------------------------------------------------------------------
  研修履歴
----------------------------------------------------------------------

社員が受講した研修・学習の記録を管理する画面です。

  登録するには:
    1. 社員を選択
    2. 研修/学習名を入力
    3. 種別（社内研修/外部研修/e-learning/OJT/自己学習/資格取得）を選択
    4. 関連スキルがあれば選択
    5. 受講日、時間数、費用、提供元、評価/感想を入力
    6. 「登録」ボタンを押す

  履歴確認:
    下部の「研修履歴一覧」で社員ごとの履歴を確認できます。
    社員を未選択のまま「表示」を押すと全員分を表示します。


----------------------------------------------------------------------
  スキル検索
----------------------------------------------------------------------

条件を指定して社員のスキル保有状況を横断検索する画面です。

  使い方:
    1. スキル名（部分一致）、最低レベル、部署、カテゴリを
       必要な条件だけ入力
    2. 「検索」ボタンを押す
    3. 検索結果が下に表示されます

  活用例:
    - 「Python レベル3以上」で検索 → プロジェクトメンバー候補を探す
    - 部署を指定して検索 → 部署のスキル保有状況を確認
    - カテゴリで検索 → 特定分野の人材を横断的に把握

  ポイント:
    - 検索ログは自動記録されます（分析用）


----------------------------------------------------------------------
  マスタ管理
----------------------------------------------------------------------

システムの基本データを管理する画面です。
上部のサブタブで切り替えます。

  スキルカテゴリマスタ:
    スキルを分類するカテゴリを管理します。
    例: プログラミング言語、フレームワーク、業務知識 など

  スキル項目マスタ:
    個々のスキルを管理します。
    カテゴリへの紐付け、レベル1〜5の定義、重要度、
    陳腐化リスクなどを設定できます。

  資格マスタ:
    管理対象の資格を登録します。
    有効期限(年)を設定すると、資格登録時に自動で期限計算されます。
    取得支援の有無、報奨金なども記録できます。

  社員マスタ:
    社員情報を管理します。
    上長IDを設定すると上長評価が使えるようになります。
    メールアドレスを設定するとアラート通知が届きます。

  職種別必須スキル定義:
    職種・等級ごとに必要なスキルと必須レベルを定義します。
    ここで定義した内容がスキルギャップ分析の基準になります。

  アラート設定:
    資格期限切れなどのアラート条件を管理します。
    条件: 資格期限 / スキル未更新 / 属人化検出 / ギャップ検出

  各マスタ共通操作:
    追加: 「+ 追加」ボタン → フォーム入力 → 追加
    編集: 一覧の「編集」ボタン → フォーム修正 → 保存


----------------------------------------------------------------------
  レポート
----------------------------------------------------------------------

集計データを閲覧する画面です。
上部のサブタブでレポートを切り替えます。

  チームスキルマップ:
    チーム×スキルごとの保有者数、カバー率、平均レベル、
    属人化リスクを表示します。

  スキルギャップ:
    社員ごとに必須レベルと現在レベルの差を表示します。
    ギャップ種別（不足/充足/超過）、優先度、推奨アクションも確認可。

  部署別サマリー:
    月次の部署別統計データを表示します。
    人数、1人あたりスキル数、平均レベル、育成計画進捗率など。

  スキル別保有者:
    スキルごとの全社保有者数、レベル分布、
    属人化リスク、育成優先度を表示します。

  資格アラート:
    資格の期限切れ・期限接近アラートを一覧表示します。

  集計データの更新:
    「全集計更新」ボタンを押すと最新データに更新されます。
    集計はトリガーでも自動実行されます（日次/週次/月次/四半期）。


----------------------------------------------------------------------
  自動処理（トリガー）
----------------------------------------------------------------------

バックグラウンドで定期実行される処理があります。

  日次: 資格期限のアラートチェック
  週次: チームスキルマップ集計、スキル別保有者サマリー集計
  月次: 部署別サマリー集計、スキルギャップ再計算
  四半期: 全集計の一括更新

  トリガーの設定:
    GASエディタからスクリプトを開き、setupTriggers()を一度実行
    してください。以降は自動で動きます。

  メール通知:
    資格期限アラート検出時にGmailApp経由で本人・上長に通知。
    社員マスタにメールアドレスが登録されている必要があります。


----------------------------------------------------------------------
  運用の流れ（推奨）
----------------------------------------------------------------------

  初期セットアップ:
    1. マスタ管理 → スキルカテゴリマスタにカテゴリを登録
    2. マスタ管理 → スキル項目マスタにスキルを登録（レベル定義も）
    3. マスタ管理 → 資格マスタに管理対象の資格を登録
    4. マスタ管理 → 社員マスタに社員を登録（上長ID・メールも設定）
    5. マスタ管理 → 職種別必須スキル定義で必須スキルを定義
    6. GASエディタで setupTriggers() を実行

  日常運用:
    - 社員がスキル登録で自己評価を登録
    - 上長が上長評価で部下のスキルを評価
    - 資格取得時に資格登録で記録
    - 研修受講後に研修履歴に記録

  定期確認:
    - ダッシュボードでKPI・アラートを確認
    - レポートで集計データを分析
    - スキルギャップの大きい社員に育成計画を作成


----------------------------------------------------------------------
  よくある質問
----------------------------------------------------------------------

  Q: スキルのレベル定義はどこで設定しますか？
  A: マスタ管理 → スキル項目マスタの編集画面で
     レベル1〜5の定義を入力できます。

  Q: 確定レベルが表示されません
  A: 上長評価がまだ行われていません。
     上長評価画面で評価を完了してください。

  Q: アラートメールが届きません
  A: 社員マスタにメールアドレスが登録されているか確認してください。
     また setupTriggers() が実行済みか確認してください。

  Q: 集計データが古いままです
  A: レポート画面の「全集計更新」ボタンで最新化できます。

  Q: テストデータを入れたい
  A: ダッシュボード下部の「仮データ投入」ボタンで
     サンプルデータを一括投入できます。

  Q: データを全部消してやり直したい
  A: ダッシュボード下部の「全データ初期化」ボタンで
     全シートのデータを削除できます（ヘッダー行は残ります）。
