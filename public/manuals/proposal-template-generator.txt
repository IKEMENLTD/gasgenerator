================================================================================
提案書テンプレート差し込み生成 使い方説明書
================================================================================

■ このシステムでできること
================================================================================

・Googleドキュメントの提案書テンプレートに顧客名・金額・担当者名などを
　自動で差し込み、提案書を一括生成できます
・生成と同時にGoogle Driveの共有リンクを自動発行します
・顧客・担当者・テンプレートをマスタ管理し、入力ミスを防ぎます
・ダッシュボードで生成状況・金額推移・期限管理を可視化します
・有効期限が近い提案書を担当者にメールで自動通知します
・毎日の生成件数・エラー件数を管理者にメールでレポートします


■ 画面構成（5タブ）
================================================================================

1. ダッシュボード　　… 生成状況の統計・グラフ表示
2. 提案書生成　　　　… 提案書依頼の登録・編集・生成実行・CSV出力
3. テンプレート管理　… テンプレートの登録・編集・削除
4. 顧客マスタ　　　　… 顧客情報の登録・編集・削除
5. 担当者マスタ　　　… 担当者情報の登録・編集・削除


■ 初回セットアップ（推奨順序）
================================================================================

以下の順番でマスタデータを登録すると、提案書生成がスムーズです。

1. 担当者マスタに自社の担当者を登録
2. 顧客マスタに顧客情報を登録
3. テンプレート管理にGoogleドキュメントのテンプレートを登録
4. 提案書生成で依頼を作成→生成実行


■ 各タブの使い方
================================================================================


【1. ダッシュボード】
--------------------------------------------------------------------------------

提案書生成タブのデータを集計して表示します。

表示項目:
・本日の生成件数　　　… 今日生成完了した提案書の件数
・有効期限切れ・期限迫り… 有効期限が3日以内または超過した件数
・当月依頼件数　　　　… 今月の全依頼件数（全ステータス合計）
・生成ステータス別件数　… 当月の未生成/生成中/生成完了/生成エラーの円グラフ
・担当者別生成件数　　… 当月の担当者ごとの依頼件数（縦棒グラフ）
・月別提案金額合計　　… 月ごとの提案金額（税抜）合計の推移（折れ線グラフ）


【2. 提案書生成】
--------------------------------------------------------------------------------

提案書の依頼登録・生成実行を行うメイン画面です。

◆ 新規依頼の登録
  1.「+ 新規依頼」ボタンをクリック
  2. フォームに以下を入力:
     - 担当者（ドロップダウンから選択）
     - 顧客（ドロップダウンから選択）
     - 使用テンプレート（標準提案書/簡易提案書/上位プラン提案書/カスタム）
     - 提案タイトル（必須）
     - 提案金額（税抜）（必須）
     - 消費税率（8% または 10%）
     - 提案期間（例: 2026年4月〜6月）
     - 提案有効期限（デフォルトは30日後）
     - 閲覧権限（リンク閲覧可 / 特定ユーザーのみ）
     - カスタム差し込み1〜3（任意）
     - 備考（任意）
  3.「保存」をクリック
  ※ 担当者名・顧客会社名・顧客担当者名・敬称・税込金額は自動で設定されます

◆ 提案書の生成
  1. テーブルの「生成」ボタンをクリック
  2. 差し込みプレビュー画面で全15変数の差し込み内容を確認
  3. 問題なければ「確定生成」をクリック
  4. Googleドキュメントが自動生成され、共有リンクが発行されます
  5. テーブルの「開く」で生成ドキュメントを確認、「コピー」でリンクをコピー

◆ 再生成
  生成完了済みの提案書も「再生成」ボタンで再度生成できます。
  内容を編集した後に新しいドキュメントを作り直したい場合に使用します。
  ※ 既存のドキュメントはそのまま残ります

◆ 編集
  テーブルの「編集」ボタンで依頼内容を修正できます。

◆ 削除
  テーブルの「削除」ボタンで依頼を削除します。確認ダイアログが表示されます。

◆ フィルタ・検索
  - ステータスフィルタ: 全ステータス/未生成/生成中/生成完了/生成エラー
  - 検索: テキスト入力で全項目を横断検索

◆ CSV出力
  「CSV出力」ボタンでフィルタ適用中のデータをCSVファイルとしてダウンロードします。
  ファイル名: proposals_YYYYMMDD.csv

◆ ステータスの意味
  - 未生成　　… 依頼登録済み、まだ生成していない
  - 生成中　　… 生成処理を実行中
  - 生成完了　… 生成成功、共有リンク発行済み
  - 生成エラー… 生成に失敗（テンプレート未設定等）。修正後に再度生成可能


【3. テンプレート管理】
--------------------------------------------------------------------------------

Googleドキュメントのテンプレートを管理します。

◆ 新規追加
  1.「+ 新規追加」ボタンをクリック
  2. 以下を入力:
     - テンプレート名（標準提案書/簡易提案書/上位プラン提案書/カスタム）
     - GoogleドキュメントID（テンプレート本体のドキュメントID）
     - 保存先フォルダID（生成ファイルの保存先DriveフォルダID、空欄ならルート）
     - ファイル名規則（例: 提案書_{会社名}_{依頼日}）
     - 有効フラグ（有効/無効）
     - 差し込み変数一覧（使用中の変数をカンマ区切り）
     - 備考
  3.「保存」をクリック

◆ テンプレートの作り方
  1. Googleドキュメントで提案書のひな形を作成
  2. 差し込みたい箇所に以下の変数を記入:
     {会社名} {敬称} {担当者名} {提案タイトル}
     {提案金額_税抜} {提案金額_税込} {消費税率}
     {提案期間} {有効期限} {依頼日}
     {担当者氏名} {担当者署名}
     {custom1} {custom2} {custom3}
  3. ドキュメントのURLからIDをコピー
     （例: https://docs.google.com/document/d/【このIDをコピー】/edit）
  4. テンプレート管理に登録

◆ 編集・削除
  テーブルの「編集」「削除」ボタンから操作します。

◆ フィルタ
  有効/無効でフィルタできます。


【4. 顧客マスタ】
--------------------------------------------------------------------------------

提案先の顧客情報を管理します。

◆ 新規追加
  1.「+ 新規追加」ボタンをクリック
  2. 以下を入力:
     - 顧客会社名（必須）
     - 顧客敬称（御中/様/ご担当者様）
     - 顧客区分（法人/個人事業主/個人）
     - 担当者名（先方の担当者）
     - メールアドレス（「特定ユーザーのみ」共有時の送信先）
     - 電話番号
     - 業種（制作/コンサル/IT/製造/小売/サービス/その他）
     - 有効フラグ（有効/無効）
     - 備考
  3.「保存」をクリック

◆ 編集・削除・検索
  テーブルの「編集」「削除」ボタン、フィルタ・検索欄から操作します。


【5. 担当者マスタ】
--------------------------------------------------------------------------------

自社の担当者情報を管理します。

◆ 新規追加
  1.「+ 新規追加」ボタンをクリック
  2. 以下を入力:
     - 氏名（必須）
     - メールアドレス（有効期限アラートの送信先）
     - 部署（営業/コンサル/管理/その他）
     - 役割（管理者/担当者/閲覧者）
       ※「管理者」は日次レポートメールの送信先になります
     - 有効フラグ（有効/無効）
     - 署名テキスト（提案書フッターに差し込まれます）
  3.「保存」をクリック

◆ 編集・削除・検索
  テーブルの「編集」「削除」ボタン、フィルタ・検索欄から操作します。


■ 自動通知メール
================================================================================

◆ 有効期限アラート（毎日 8:00）
  有効期限が3日以内または超過している提案書（生成完了のもの）の担当者に
  メールで通知します。

◆ 日次生成レポート（毎日 9:00）
  前日の生成完了件数とエラー件数を、「管理者」役割の担当者にメール送信します。


■ 閲覧権限について
================================================================================

提案書生成時に選択した閲覧権限に応じて共有設定が自動適用されます。

・リンク閲覧可　　… リンクを知っている全員が閲覧可能
・特定ユーザーのみ… 顧客マスタのメールアドレスに閲覧権限を付与


■ スマートフォン対応
================================================================================

スマートフォンやタブレットからもご利用いただけます。
左上のハンバーガーメニュー（≡）からタブを切り替えてください。
テーブルは横スクロールで全列を確認できます。
