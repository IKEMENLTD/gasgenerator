面接評価統一シート 使い方ガイド
========================================

左側のメニューから各画面に移動できます。
スマホの場合は左上の「≡」ボタンでメニューが開きます。

■ ダッシュボード
────────────────────────────────
ログイン後に最初に表示される画面です。

  - 候補者数、評価件数、平均スコア、面接官数が上部に表示されます
  - 選考ステータス分布・判定分布・職種別候補者数のグラフで
    採用活動の全体像を一目で確認できます
  - 面接官間で評価スコアに大きな差がある候補者がいる場合、
    「評価ブレ検知」としてアラートが表示されます
    クリックすると対象者の詳細を確認できます


■ 候補者管理
────────────────────────────────
応募者の登録・管理を行う画面です。

【候補者を登録する】
  1.「+ 新規候補者」ボタンを押す
  2. 氏名と応募職種（必須）を入力
  3. 応募経路・現職/前職・経験年数などを必要に応じて入力
  4. 選考ステータスを選択（初期値は「書類選考」）
  5.「保存」ボタンを押す

【候補者情報を編集する】
  テーブルの「編集」ボタンから情報を更新できます。
  選考が進んだらステータスを更新してください。

【候補者を検索・絞り込む】
  - 上部のドロップダウンでステータス別に絞り込み
  - 検索欄に名前・職種などを入力してフリーワード検索


■ 面接評価入力
────────────────────────────────
面接後の評価を入力する画面です。

【評価を登録する】
  1.「+ 新規評価」ボタンを押す
  2. 候補者・面接ステージ・面接官名（必須）を入力
  3. 各評価項目のスコア（1〜5）を選択
     ※ 評価項目は「評価項目マスタ」で設定した内容が表示されます
     ※ 各点数にカーソルを合わせると定義が表示されます
  4. 総合判定を選択（強く推薦 / 推薦 / 保留 / 不推薦）
  5. 良かった点・懸念点・質問メモ・申し送りを必要に応じて入力
  6.「保存」ボタンを押す

  登録すると総合スコア（加重平均）が自動計算されます。
  同じ候補者に複数の面接官が評価ブレのある評価を入力した場合、
  通知が表示されます。

【過去の評価を確認する】
  テーブルの「詳細」ボタンで各評価の全項目を確認できます。
  検索欄で候補者名・面接官名などで絞り込みもできます。


■ 候補者別サマリー
────────────────────────────────
候補者ごとの評価結果をまとめた画面です。

  - 平均スコア・最高/最低スコア・ブレ幅が一覧で確認できます
  - 推薦数と不推薦数から最終判定が自動で算出されます
  - ブレ幅が大きい（1.5点以上）候補者はバッジの色で示されます
    黄色: 1.0〜2.0点の差 / 赤: 2.0点以上の差
  - 「分析更新」ボタンで最新データに更新されます


■ 面接官別分析
────────────────────────────────
面接官ごとの評価傾向を分析する画面です。

  - 各面接官の担当数・平均スコア・標準偏差・推薦率が表示されます
  - 評価傾向が「甘め」「普通」「厳しめ」で自動判定されます
    全体平均より+0.3以上 → 甘め / −0.3以下 → 厳しめ
  - グラフで面接官間のスコア比較を視覚的に確認できます
  - 面接官間の評価基準のすり合わせに活用してください


■ 評価項目マスタ
────────────────────────────────
面接で使う評価項目を設定する画面です。

【評価項目を追加する】
  1.「+ 評価項目追加」ボタンを押す
  2. カテゴリを選択（スキル / コミュニケーション / カルチャー / ポテンシャル / その他）
  3. 評価項目名を入力（例: 「技術力」「論理的思考」など）
  4. 1〜5点の各点数の定義を入力（任意）
     ※ ここで入力した内容が評価入力時のツールチップに表示されます
  5. 重みを入力（加重平均に使われます。未設定なら均等配分）
  6. 対象職種を選択（「全職種」または特定の職種名）
  7.「保存」ボタンを押す

【評価項目を編集・削除する】
  テーブルの「編集」「削除」ボタンから操作できます。
  ※ 削除しても過去の評価データには影響しません


■ 採用結果振り返り
────────────────────────────────
入社後の評価と面接時の評価を比較する画面です。

【振り返りを登録する】
  1.「+ 振り返り登録」ボタンを押す
  2. 候補者を選択
  3. 入社日を入力
  4. 入社後評価を選択（非常に良い / 良い / 普通 / やや課題あり / 課題が多い）
  5. 面接評価との一致度を選択（一致 / 概ね一致 / やや乖離 / 乖離あり）
  6. 振り返りコメントを入力（任意）
  7.「保存」ボタンを押す

  散布図で面接時スコアと入社後評価の関係を確認でき、
  評価基準の精度向上に役立ちます。


■ バッジの色について
────────────────────────────────
  各テーブルのバッジは色で状態を表しています。

  青   … 最も良い状態（強く推薦、内定、ブレ小、一致 など）
  緑   … 良い状態（推薦、概ね一致 など）
  黄   … 注意が必要（保留、やや乖離、甘め傾向 など）
  赤   … 対応が必要（不推薦、不採用、厳しめ傾向 など）
  グレー … 終了・対象外（辞退 など）


■ その他の便利機能
────────────────────────────────
  - 各テーブルは10件ごとにページ分けされます
  - モーダル（入力画面）は ×ボタン / キャンセル / Escキー /
    背景クリック のいずれかで閉じられます
  - スマホでも全機能を利用できます
  - 週次レポートがメールで自動配信されます（毎週月曜9時）
