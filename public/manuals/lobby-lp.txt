============================================================
  Lobby LP 使い方ガイド
============================================================

Lobby LP は、面接に来る候補者に向けて
「おすすめ記事」と「当日の案内情報」をまとめた
Webページをかんたんに発行できるツールです。

発行したページのURLを候補者にメールで送るだけで、
面接前に読んでほしい記事や会場案内を届けられます。


------------------------------------------------------------
  画面の構成
------------------------------------------------------------

上部のナビゲーションから5つの画面を切り替えて使います。

  ダッシュボード … 発行済みページの一覧・管理
  発行           … 新しい候補者ページを作る
  コンテンツ     … おすすめ記事の登録・編集
  セット         … 記事の組み合わせテンプレート
  会場           … 面接会場の情報管理


============================================================
  はじめに準備すること
============================================================

ページを発行する前に、以下の3つを登録しておきます。
一度登録すれば、繰り返し使えます。

  1. コンテンツ（おすすめ記事）
  2. セット（記事の組み合わせ）
  3. 会場（面接場所の情報）

登録の順番は 1 → 2 → 3 がおすすめです。
セットを作るときにコンテンツを選ぶので、
先にコンテンツを登録しておくとスムーズです。


============================================================
  コンテンツ（おすすめ記事）を登録する
============================================================

候補者に読んでほしい記事を登録する画面です。
採用ブログ、会社紹介、社員インタビューなど、
候補者に届けたいWebページを自由に登録できます。

■ 登録のしかた

  1.「コンテンツ」画面を開く
  2.「+ 新規記事を追加」をクリック
  3. 以下を入力して「保存する」

    タイトル（必須）   … 候補者に表示される記事タイトル
    URL（必須）        … 記事のURL
    サムネイル画像URL  … 記事と一緒に表示されるサムネイル
    掲載元ラベル       … 「note」「コーポレートサイト」など
    おすすめ理由（必須）… 「なぜこの記事を読んでほしいか」
                         60字以内を目安に書いてください
    カテゴリ           … 記事の分類（自由入力）

  ※ サムネイル画像URLの調べ方:
    記事のページをブラウザで開く → 右クリック →
    「ページのソースを表示」 → og:image で検索 →
    content="..." のURLをコピー

■ 編集・無効化

  各記事のカードにある「編集」から内容を修正できます。
  「無効化」すると、セット作成時の選択肢に表示されなくなります。
  （削除ではないので、あとから「有効化」で戻せます）

■ カテゴリでの絞り込み

  記事が増えてきたら、画面上部のカテゴリ選択で
  表示を絞り込めます。


============================================================
  セット（記事の組み合わせ）を作る
============================================================

「この候補者にはこの記事セットを送る」という
テンプレートを作っておく機能です。
例えば「エンジニア向け」「営業向け」など、
ポジション別にセットを作っておくと便利です。

■ 作成のしかた

  1.「セット」画面を開く
  2.「+ 新規セットを作成」をクリック
  3. 以下を入力して「保存する」

    セット名（必須）… 「エンジニア基礎セット」など
    対象メモ        … 「新卒エンジニア候補者向け」など
    デフォルト会場  … このセットで発行するとき
                      自動的に選ばれる会場
    記事            … 2〜5本を選択（順番が表示順になります）

  ※ 「+ 記事を追加」で記事枠を増やせます
  ※ 各記事の「×削除」で記事枠を減らせます
  ※ 「おすすめ理由の上書き」欄に入力すると、
     コンテンツに登録した理由の代わりにその文が表示されます


============================================================
  会場（面接場所）を登録する
============================================================

候補者に伝える面接会場の情報を管理する画面です。

■ 登録のしかた

  1.「会場」画面を開く
  2.「+ 新規会場を追加」をクリック
  3. 以下を入力して「保存する」

    会場名（必須）        … 「本社オフィス」など
    住所（必須）          … 住所テキスト
    Google MapsリンクURL  … 地図リンク（候補者ページに
                            「Google Mapsで開く」ボタンが出ます）
    行き方テキスト        … 「渋谷駅ハチ公口から徒歩5分…」など
    行き方動画URL         … YouTube動画がある場合はそのURL
    緊急連絡先電話番号（必須）… 当日困ったとき用の電話番号
    緊急連絡先補足        … 「受付は9F、担当:田中」など
    デフォルト会場にする  … チェックすると、
                            発行画面で自動選択されます

  ※ デフォルト会場は1つだけ設定できます


============================================================
  ページを発行する
============================================================

準備ができたら、候補者ごとにページを発行します。

■ 発行のしかた

  1.「発行」画面を開く
  2. 以下を入力

    候補者の姓（必須）  … 「田中」など
    候補者の名          … 「太郎」など（任意）
    応募ポジション      … 「エンジニア」など（任意）
    おすすめセット（必須）… 登録済みセットから選択
    会場（必須）        … 登録済み会場から選択
    面接日              … 面接予定日
    候補者へのメッセージ … ページ上部に表示される文
    発行者名            … 自分の名前（管理用）

  3.「ページを発行する」をクリック

■ 記事を個別に変更したいとき

  「▶ 記事を個別に変更する」を開くと、
  セットに含まれる記事を1本ずつ差し替えられます。
  「この候補者にはセットの3番目だけ別記事にしたい」
  といったときに使います。

■ 発行後の画面

  発行が完了すると、以下が表示されます。

  ・候補者ページのURL
  ・「URLをコピー」ボタン
  ・メール例文（URLを含む送信用テキスト）
  ・「例文ごとコピー」ボタン
  ・「候補者ページを確認する」リンク
  ・「続けて発行」ボタン（別の候補者を続けて発行）

  URLをコピーして、候補者へのメールに貼り付けてください。


============================================================
  ダッシュボード（発行済みページの管理）
============================================================

発行したページの状態を一覧で確認できます。

■ 一覧に表示される情報

  候補者     … 候補者の名前
  ポジション … 応募ポジション
  セット     … 使用したセット名
  面接日     … 面接予定日
  閲覧       … 候補者がページを開いた回数
  記事クリック … クリックされた記事数 / 全記事数
  ステータス … active（有効）または expired（期限切れ）

■ 操作

  「URL」ボタン  … そのページのURLをクリップボードにコピー
  「削除」ボタン … ページを削除（候補者がアクセスできなくなります）

■ ページネーション

  11件以上あると「前 / 次」ボタンで
  ページを切り替えられます（10件ずつ表示）。

■ ステータスの自動更新

  面接日から3日が経過すると、ステータスが
  自動的に「expired」に変わります。
  期限切れのページにアクセスした候補者には
  「ご利用期間が終了しました」と表示されます。


============================================================
  候補者に届くページについて
============================================================

候補者がURLを開くと、以下の内容が表示されます。

  ・あいさつ文とメッセージ
  ・おすすめ記事の一覧
    （サムネイル、タイトル、掲載元、おすすめ理由）
  ・面接会場の案内
    （会場名、住所、地図ボタン、行き方、動画）
  ・緊急連絡先（電話ボタン付き）

候補者が記事をクリックすると、
ダッシュボードの「記事クリック」に反映されます。
「3/4」と表示されていれば、
4本中3本の記事がクリックされたということです。


============================================================
  よくある質問
============================================================

Q. 発行したあとに記事やメッセージを変えたい
A. 発行済みページの内容は直接編集できません。
   一度削除して、新しく発行し直してください。

Q. セットの記事構成を変えたら、発行済みページも変わる？
A. 変わりません。発行時点の記事がコピーされるので、
   あとからセットを変更しても影響しません。

Q. 同じ候補者に2回ページを発行できる？
A. できます。それぞれ別のURLが発行されます。
   古いほうが不要なら、ダッシュボードから削除してください。

Q. ページの有効期限は？
A. 面接日から3日間です。面接日を入力しなかった場合は
   期限切れになりません。

Q. スマホからも操作できる？
A. はい。管理画面・候補者ページともにスマホ対応しています。
