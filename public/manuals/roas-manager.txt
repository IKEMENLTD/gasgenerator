========================================
 広告費対効果（ROAS）管理システム
 使い方説明書
========================================

■ 概要
広告媒体ごとの広告費と売上を管理し、ROAS（広告費対効果）を
自動計算・分析するシステムです。
媒体別・キャンペーン別の日次/月次分析、効率ランキング、
低効率アラート、予算配分シミュレーション、LTV分析、
トレンド分析を搭載しています。

■ アクセスURL
https://script.google.com/macros/s/AKfycbyPSxoqwhmU22wx81WtYGwb5_gmxvbkoJIsk8-isLPPOSTIV1UngF1UQoZOJNlxYjpy/exec

========================================
 初回セットアップ
========================================

1. 上記URLにアクセスする
2. 左メニュー「設定」を開く
3. 「シート初期化」ボタンを押す
   → 全15シートのヘッダーが自動作成されます
4. 「トリガー設定」ボタンを押す
   → 日次/週次/月次の自動計算トリガーが設定されます

========================================
 基本操作
========================================

【媒体管理】
  広告配信先の媒体を登録・管理します。
  1. 左メニュー「媒体管理」を開く
  2. 「+ 媒体追加」ボタンを押す
  3. 媒体名、種別（リスティング/ディスプレイ/SNS/動画/アフィリエイト）、
     課金形態（CPC/CPM/CPA/固定）、目標ROAS、目標CPA、
     月間予算、担当者を入力
  4. 「保存」を押す

【キャンペーン管理】
  各媒体に紐づくキャンペーンを登録します。
  ※ 先に媒体を登録してください。
  1. 左メニュー「キャンペーン管理」を開く
  2. 「+ キャンペーン追加」ボタンを押す
  3. キャンペーン名、媒体（プルダウン選択）、目的、ターゲット、
     期間、月間予算、目標ROAS、目標CPA、担当者を入力
  4. 「保存」を押す

【広告費データ登録】
  日次の広告費実績を登録します。
  ※ 先に媒体・キャンペーンを登録してください。
  1. 左メニュー「広告費データ」を開く
  2. 「+ 新規登録」ボタンを押す
  3. 日付、媒体、キャンペーン、広告費、表示回数、
     クリック数、CV数を入力
     （CTR/CPC/CVR/CPAは自動計算されます）
  4. 「保存」を押す

  ※ フィルター機能で期間・媒体を絞り込めます。

【売上データ登録】
  広告経由の売上を登録します。
  1. 左メニュー「売上データ」を開く
  2. 「+ 新規登録」ボタンを押す
  3. 売上日時、流入元媒体、キャンペーン、顧客ID、
     顧客種別（新規/既存）、注文ID、売上金額、原価、
     商品カテゴリ、UTM、デバイスを入力
     （粗利は自動計算されます）
  4. 「保存」を押す

========================================
 分析機能
========================================

【ダッシュボード】
  トップページ。今月のKPIを一覧表示します。
  - 広告費合計 / 売上合計 / ROAS / CV数 / 粗利 / 新規顧客数
  - 媒体別ROAS比較グラフ（棒グラフ）
  - 広告費構成比（ドーナツグラフ）
  - ROAS推移（折れ線グラフ・過去30日）
  - 効率TOP5 / WORST5
  - 未対応アラート一覧

【ROAS分析】
  6つのタブで詳細分析できます。
  - 媒体別（日次）: 日ごとの媒体別ROAS・判定
  - CP別（日次）: 日ごとのキャンペーン別ROAS
  - 媒体別（月次）: 月ごとの達成率・前月比・予算消化率・推奨アクション
  - CP別（月次）: 月ごとのキャンペーン効率ランク
  - LTV分析: 媒体/CP別の顧客LTV・LTV-ROAS・リピート率
  - トレンド: 4週移動平均ROAS・トレンド判定

  ※「再計算」ボタンで最新データから再集計できます。

【ランキング】
  全キャンペーンをROAS順にランク付けします。
  - 「ランキング更新」ボタンで最新データから生成
  - 順位、ROAS、CPA、前月順位、変動、推奨アクションを表示

【アラート】
  設定した条件を下回るキャンペーンを自動検出します。
  - 「アラートチェック実行」で手動チェック
  - 「対応」ボタンで状態を「未対応→検討中→対応済」に変更可能
  - タブで未対応/検討中/対応済をフィルタリング

【予算最適化】
  ROAS実績に基づいて予算配分をシミュレーションします。
  1. シナリオ名を入力（デフォルト: ROAS比例配分）
  2. 総予算を入力
  3. 「シミュレーション実行」を押す
  → 現状配分 vs 提案配分の比較表が表示されます

========================================
 設定
========================================

【アラート設定】
  アラート条件を追加・編集できます。
  - 条件: ROAS / CPA / 予算消化 / CV数
  - 対象: 全媒体 / 特定媒体 / 特定キャンペーン
  - 頻度: 毎日 / 週次
  - 通知先メールアドレスを設定

【自動処理】
  - トリガー設定: 日次朝計算、日次夕アラート、
    週次レポート、月次レポートの自動実行を設定
  - シート初期化: 全15シートのヘッダーを作成
  - 全計算実行: 全分析を一括再計算

========================================
 スプレッドシート構成（全15シート）
========================================

 1. 広告媒体マスタ      - 媒体の基本情報・目標値
 2. キャンペーンマスタ  - キャンペーンの基本情報
 3. 広告費データ        - 日次の広告費実績
 4. 売上データ          - 広告経由の売上実績
 5. 媒体別ROAS（日次）  - 日次の媒体別ROAS自動集計
 6. キャンペーン別ROAS（日次） - 日次のCP別ROAS自動集計
 7. 媒体別ROAS（月次）  - 月次の媒体別ROAS自動集計
 8. キャンペーン別ROAS（月次） - 月次のCP別ROAS自動集計
 9. 広告効果ランキング  - キャンペーン効率ランキング
10. 低効率広告アラート  - アラート検出履歴
11. 予算配分シミュレーション - シミュレーション結果
12. LTV連携分析        - 顧客LTV分析結果
13. トレンド分析        - 4週移動平均・トレンド
14. アラート設定        - アラート条件マスタ
15. ダッシュボード      - KPIサマリー

========================================
 自動処理（トリガー）
========================================

 - 毎日 8:00  日次ROAS計算（媒体別・CP別）
 - 毎日 18:00 アラートチェック・メール通知
 - 毎週月曜   週次レポートメール送信
 - 毎月 1日   月次ROAS計算・月次レポートメール送信

 ※ 設定画面の「トリガー設定」ボタンで有効化されます。

========================================
 注意事項
========================================

 - 媒体・キャンペーンを先に登録してから
   広告費・売上データを登録してください。
 - データ登録後、分析シートへの反映は
   自動トリガーまたは手動の「再計算」で行われます。
 - ROAS = 売上 / 広告費 x 100（%）で計算されます。
 - 目標ROASを100%以上に設定すると
   広告費以上の売上が目標となります。
 - アラートの通知先メールはGoogleアカウントの
   メールアドレスから送信されます。
