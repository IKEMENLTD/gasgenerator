仕入先・外注先情報管理 使い方説明書

====================================
1. システム概要
====================================

仕入先・外注先の基本情報、契約条件、取引履歴、評価を一元管理するシステムです。
担当者不在時でも業者情報にアクセスでき、過去の取引条件や評価を全員で共有できます。

対象ユーザー: 総務部門、購買部門

====================================
2. アクセス方法
====================================

GASウェブアプリのURLをブラウザで開いてください。
初回アクセス時にGoogleアカウントでのログインが必要です。

====================================
3. 画面構成（10タブ）
====================================

左サイドバーから各タブに切り替えます。

【メインメニュー】
  - ダッシュボード ... 全体概況・アラート・グラフ
  - 業者マスタ ...... 業者の基本情報管理
  - 担当者情報 ...... 業者側の担当者管理
  - 取引条件 ........ 契約・支払条件の管理
  - 取扱商品 ........ 商品・サービスの詳細管理

【記録・評価】
  - 取引履歴 ........ 発注・見積等の記録
  - 評価管理 ........ 業者の定期評価
  - やり取り履歴 .... 電話・メール等の記録

【分析・設定】
  - 比較・分析 ...... 業者比較表・カテゴリサマリー
  - 設定 ............ カテゴリ管理・契約更新・トリガー

====================================
4. 初期設定（最初に1回だけ）
====================================

(1) 設定タブ > カテゴリマスタ で業種カテゴリを登録
    例: 印刷、IT、清掃、設備、物流、製造
    - 「+ 新規カテゴリ」ボタンで追加
    - カテゴリ名、説明、主な用途、担当部署を入力
    - ステータスは「有効」のまま保存

(2) 設定タブ > トリガー設定 で自動処理を確認
    - dailyContractCheck: 毎日9:00 契約更新期限チェック
    - weeklyUpdateSummary: 毎週月曜8:00 カテゴリ別サマリー更新
    - monthlyReport: 毎月1日8:00 月次レポート生成
    ※ トリガーは初回アクセス時に自動設定されます

====================================
5. 基本操作
====================================

■ データの登録
  各タブの「+ 新規登録」ボタンをクリック
  → フォームに入力 → 「保存」ボタン
  ※ 必須項目（*マーク）は入力必須です

■ データの編集
  一覧の「編集」ボタンをクリック
  → フォームに既存データが表示 → 修正して「保存」

■ データの削除
  一覧の「削除」ボタンをクリック
  → 確認ダイアログで「OK」を選択

■ データの検索・絞り込み
  各タブ上部のドロップダウンや検索欄を使用
  - ドロップダウン: ステータス、カテゴリ、業者等で絞り込み
  - 検索欄: キーワードで部分一致検索

====================================
6. 主要機能の使い方
====================================

--------------------------------------
6-1. 業者の登録（業者マスタ）
--------------------------------------
「+ 新規登録」をクリックして以下を入力:
  - 業者名（必須）、業者名（カナ）
  - 業者種別: 仕入先 / 外注先 / 両方
  - 業種カテゴリ: 登録済みカテゴリから選択
  - 住所、電話番号、FAX、Webサイト
  - 法人番号、設立年、従業員規模
  - 当社担当者名
  - ステータス: 取引中 / 休止 / 取引終了 / 候補

自動処理:
  - 法人番号・電話番号の重複チェック（新規登録時）
  - 登録日・最終更新日の自動設定
  - カテゴリの登録業者数を自動更新

--------------------------------------
6-2. 担当者の登録（担当者情報）
--------------------------------------
業者側の担当者を登録します。
  - 業者を選択（必須）→ 業者名が自動補完
  - 担当者名（必須）、部署、役職
  - 電話番号（直通）、携帯番号、メールアドレス
  - 対応可能時間、担当領域
  - メイン担当フラグ: ○ で主担当を明示

--------------------------------------
6-3. 契約情報の登録（取引条件）
--------------------------------------
業者との契約条件を記録します。
  - 業者を選択（必須）
  - 契約種別: 基本契約 / 業務委託 / スポット / 年間契約
  - 契約開始日、契約終了日
  - 自動更新: あり / なし
  - 支払条件（例: 月末締め翌月末払い）
  - 支払方法: 振込 / 引落 / クレジット
  - 標準納期、最低発注金額

注意: 契約終了日を入力すると、90日以内の場合
ダッシュボードの「契約更新アラート」に自動表示されます。

--------------------------------------
6-4. 商品・サービスの登録（取扱商品）
--------------------------------------
業者が提供する商品やサービスを管理します。
  - 業者を選択（必須）、商品・サービス名（必須）
  - カテゴリ、単価、単位、最小発注数
  - 納期目安、品質評価（1〜5）
  - ステータス: 取扱中 / 廃止

--------------------------------------
6-5. 取引の記録（取引履歴）
--------------------------------------
発注・見積・問合せ・クレーム等を記録します。
  - 業者を選択（必須）、取引日（必須）
  - 取引種別: 発注 / 見積 / 問合せ / クレーム
  - 案件名・内容、発注金額
  - 納品日（予定）と（実績）を両方入力すると
    → 納期遵守（○/×）が自動判定

自動処理:
  - 業者マスタの最終更新日を自動更新
  - ダッシュボードの「最近の取引」「年間金額ランキング」に反映

--------------------------------------
6-6. 業者の評価（評価管理）
--------------------------------------
定期的な業者評価を記録します。
  - 業者を選択（必須）
  - 品質評価、納期評価、価格評価、対応評価（各1〜5）
  - → 総合評価が自動計算（4項目の平均）
  - 継続判断: 継続 / 条件付継続 / 見直し / 終了
  - 良い点、改善点・課題、コメント

自動処理:
  - 業者マスタの総合評価を全評価の平均で自動更新
  - 評価2以下の業者はダッシュボードにアラート表示

--------------------------------------
6-7. やり取りの記録（やり取り履歴）
--------------------------------------
業者との連絡・打合せを記録します。
  - 業者を選択（必須）
  - 日時、業者担当者、当社担当者
  - 連絡方法: 電話 / メール / 訪問 / Web会議
  - 用件、内容、次回アクション

--------------------------------------
6-8. 業者比較（比較・分析）
--------------------------------------
■ 業者比較タブ
  最大3社を比較する表を作成します。
  - 比較目的: 新規選定 / 見直し / コスト比較
  - 業者1〜3を選択 → 業者名が自動補完
  - 比較項目、比較結果サマリー、推奨業者を記入

■ カテゴリサマリータブ
  「サマリー更新」ボタンでカテゴリ別の集計を更新
  - 登録業者数、取引中業者数、平均評価
  - 年間取引金額、主要業者TOP3、新規候補数

====================================
7. ダッシュボード
====================================

トップページに以下の情報を一覧表示します。

■ サマリーカード
  - 業者総数、取引中、休止、候補、契約更新アラート数

■ グラフ
  - 業種カテゴリ別内訳（ドーナツグラフ）
  - ステータス別内訳（ドーナツグラフ）

■ アラート
  - 契約更新アラート（90日以内）: 残日数を色分け表示
  - 評価低い業者（2以下）: 該当業者を警告表示

■ 一覧
  - 最近の取引（10件）
  - 年間取引金額ランキング（TOP10）
  - 長期未取引業者（6ヶ月以上）
  - 新規登録・更新業者（30日以内）
  - 取引中業者一覧（業者名・カテゴリ・評価・電話番号）

====================================
8. 設定
====================================

■ カテゴリマスタ
  業種カテゴリの追加・編集・削除
  登録業者数は業者保存時に自動更新

■ 契約更新管理
  「更新チェック実行」で契約更新アラートを手動更新
  90日以内に終了する契約を一覧表示

■ トリガー設定
  自動トリガーの一覧を確認（初回アクセス時に自動設定）
  - 毎日9:00 ... 契約更新期限チェック
  - 毎週月曜8:00 ... カテゴリ別サマリー更新
  - 毎月1日8:00 ... 月次レポート生成

====================================
9. データの並び順
====================================

全てのデータ一覧は新しい順（ID降順）で表示されます。
取引履歴は取引日の新しい順、やり取り履歴は日時の新しい順です。

====================================
10. 注意事項
====================================

- 業者を削除すると、関連する担当者・契約・取引等は
  自動削除されません。先に関連データを整理してください。
- 総合評価は評価管理で登録した評価の平均で自動計算されます。
  業者マスタで直接変更しても、次の評価保存時に上書きされます。
- 契約更新アラートは契約終了日から90日以内が対象です。
- 法人番号・電話番号の重複チェックは新規登録時のみ行われます。

====================================
11. スプレッドシート構成（11シート）
====================================

  1. 業者マスタ .............. 業者の基本情報
  2. 業者担当者情報 .......... 業者側担当者
  3. 取引条件・契約情報 ...... 契約・支払条件
  4. 取扱商品・サービス詳細 .. 商品・サービス
  5. 取引履歴 ................ 発注・見積等の記録
  6. 業者評価履歴 ............ 定期評価
  7. 業者カテゴリマスタ ...... 業種カテゴリ
  8. 問合せ・やり取り履歴 .... 連絡記録
  9. 業者比較表 .............. 業者比較
  10. 業者サマリー ........... カテゴリ別集計
  11. 契約更新管理 ........... 契約更新アラート
