====================================================
人件費シミュレーション 使い方説明書
====================================================

----------------------------------------------------
1. 概要
----------------------------------------------------

人件費シミュレーションは、社員の人件費を総合的に管理・分析するためのWebアプリケーションです。
採用・昇給の影響を事前にシミュレーションし、予算管理や中長期予測を支援します。

こんなことに使えます:
- 新規採用時の総コスト（給与+社会保険+福利厚生+採用費用）を即座に算出
- 昇給・ベースアップ実施時の年間コスト増加額を事前把握
- 部署別・職種別・等級別の人件費分析と可視化
- 月次の予算管理と実績の乖離チェック
- 3〜5年の中長期人件費予測による経営計画支援
- 人件費率や1人あたりコストなどのKPI把握

対象ユーザー:
経営者、人事部門、経理部門、事業部長


----------------------------------------------------
2. 初回セットアップ
----------------------------------------------------

Webアプリを初めて開いた際は、ダッシュボードに「セットアップが必要です」と表示されます。

手順:
1. ダッシュボードの「シート初期化」ボタンをクリック
2. 18シートが自動作成され、ヘッダー行とサンプルデータが入ります
3. 自動で人件費計算とサマリー生成が行われます
4. ダッシュボードにデータが表示されれば完了です

または「設定」タブ > 「システム」タブからも初期化できます。


----------------------------------------------------
3. ダッシュボード
----------------------------------------------------

アプリを開くと最初に表示される画面です。

表示内容:
- 月額人件費合計: 全在籍社員の月額人件費の合計
- 在籍人数: 現在の在籍社員数
- 1人あたり月額: 月額人件費合計 / 在籍人数
- 予算達成率: 当月の人件費 / 予算額 のパーセンテージ
- 部署別人件費グラフ: 横棒グラフで部署ごとの月額コストを表示
- コスト構造グラフ: 円グラフで給与・法定福利費・福利厚生費の割合を表示
- 等級別人員分布: 棒グラフで各等級の人数を表示
- 雇用形態別人員: 円グラフで正社員・契約社員・パート等の割合を表示
- アラート一覧: 有効なアラート設定を表示
- 最近のシミュレーション: 直近5件のシミュレーション履歴

操作:
- 「更新」ボタン: ダッシュボードを最新データで再表示
- 「再計算」ボタン: 全社員の人件費を再計算し、サマリーも再生成


----------------------------------------------------
4. 社員管理
----------------------------------------------------

社員のマスタデータを管理する画面です。

機能:
- 社員一覧表示: 全社員のID、氏名、部署、役職、等級、雇用形態、給与情報を一覧表示
- 検索: 氏名またはIDで絞り込み
- 部署フィルタ: 特定の部署のみ表示
- ステータスフィルタ: 在籍/休職/退職予定で絞り込み
- 社員追加: 「社員追加」ボタンで新規登録フォームを開く
- 社員編集: 一覧の「編集」ボタンでデータを修正
- 社員削除: 一覧の「削除」ボタンで削除（確認ダイアログあり）

登録項目:
社員ID（自動生成可）、氏名、部署ID、部署名、チーム名、役職、
等級/グレード（G1〜G5）、職種、雇用形態（正社員/契約社員/パート/派遣）、
入社日、生年月日、扶養人数、基本給（月額）、役職手当、その他手当、
想定賞与（年間）、ステータス

自動計算項目:
- 月額総支給 = 基本給 + 役職手当 + その他手当
- 年収 = 月額総支給 x 12 + 想定賞与
- 年齢（生年月日から自動算出）
- 勤続年数（入社日から自動算出）


----------------------------------------------------
5. 人件費明細
----------------------------------------------------

個人別の人件費内訳を表示する画面です。

表示項目:
- 社員ID、氏名、部署、雇用形態
- 月額総支給
- 法定福利費（健康保険、厚生年金、雇用保険、労災保険、介護保険、子ども拠出金）
- 通勤手当、その他福利厚生費
- 月額人件費合計
- 年間人件費合計

操作:
- 「人件費再計算」ボタン: 全社員の人件費を法定福利費率マスタ・福利厚生費マスタの
  最新データで再計算します

計算ロジック:
- 各社会保険は「月額総支給 x 会社負担率」で算出（上限額がある場合は上限額で計算）
- 介護保険は40歳以上のみ適用
- 福利厚生費は適用条件（全員/正社員のみ/管理職のみ）に基づいて適用
- 家族手当は扶養人数に連動


----------------------------------------------------
6. 採用シミュレーション
----------------------------------------------------

新規採用時の総コストを事前に算出する画面です。

入力項目:
- シミュレーション名
- 採用予定部署、採用予定職種
- 雇用形態（正社員/契約社員/パート/派遣）
- 等級/グレード（G1〜G5）
- 想定基本給（月額）、想定役職手当、想定諸手当
- 想定賞与（年間）
- 採用人数
- 入社予定月
- 採用チャネル（エージェント/求人広告/リファラル/直接応募）

出力結果:
- 月額人件費合計（給与+社会保険+福利厚生費）
- 年間人件費合計
- 採用コスト（初期）: 求人広告費、エージェント手数料、面接費用、入社準備費、研修費、PC・備品の合計
- 初年度総コスト: 年間人件費 + 採用コスト
- 給与に対する総コスト倍率（例: 1.35倍）
- 1人あたり年間コスト

使い方:
1. 左側のフォームに条件を入力
2. 「シミュレーション実行」をクリック
3. 右側に結果が表示される
4. 下部に過去のシミュレーション履歴が一覧表示される

シミュレーション結果はスプレッドシートに自動保存されます。


----------------------------------------------------
7. 昇給シミュレーション
----------------------------------------------------

昇給・ベースアップ実施時の影響額を算出する画面です。

入力項目:
- シミュレーション名
- 対象範囲（全社/部署名を選択）
- 昇給種別（定期昇給/ベースアップ/昇格）
- 昇給率（%）または昇給額（円）
  ※100以下の場合はパーセントとして、100超の場合は円として処理
- 適用開始月

出力結果:
- 対象人数
- 現在の月額給与計
- 昇給後の月額給与計
- 月額増加額
- 年間増加額
- 法定福利費増加（年間）: 昇給による社会保険料増加分
- 総増加額（年間）: 年間増加額 + 法定福利費増加

使い方:
1. 左側のフォームに条件を入力
2. 「シミュレーション実行」をクリック
3. 右側に結果が表示される


----------------------------------------------------
8. 分析
----------------------------------------------------

人件費を様々な切り口で分析する画面です。4つのサブタブがあります。

[部署別]
部署ごとの人員数、正社員数、非正規社員数、月額給与計、法定福利費、人件費合計、
1人あたり人件費を一覧表示

[職種別]
職種ごとの人員数、平均年収、平均人件費、総人件費、構成比を一覧表示

[等級別]
等級（G1〜G5等）ごとの人員数、構成比、平均基本給、平均年収、総人件費を一覧表示

[コスト構造]
人件費の内訳比率（給与・賞与比率、法定福利費比率、福利厚生費比率）、
正社員比率、管理職比率、平均年齢、平均勤続年数を表示

操作:
- 「サマリー再生成」ボタン: 最新の社員データ・コストデータからサマリーを再生成


----------------------------------------------------
9. 予算管理
----------------------------------------------------

年間の人件費予算と実績を管理する画面です。

表示項目:
- 年度、月
- 予算（人件費）、実績（人件費）
- 差異（予算 - 実績）
- 達成率（%）
- 予算人員数、実績人員数

グラフ:
予算と実績の推移を棒グラフで表示（実績データがある月のみ）

データはスプレッドシートの「年間人件費予算管理」シートに直接入力するか、
月次の自動集計（毎月15日）で実績が更新されます。


----------------------------------------------------
10. 中長期予測
----------------------------------------------------

3〜5年の人件費推移を予測する画面です。

表示項目:
- 年度
- 期首人員数、採用予定数、退職想定数、期末人員数
- 平均昇給率（%）
- 年間人件費合計
- 前年比（%）
- 人件費率予測（%）

グラフ:
年間人件費合計の推移（折れ線）と期末人員数の推移を同時に表示

データはスプレッドシートの「中長期人件費予測」シートに直接入力します。


----------------------------------------------------
11. マスタ設定
----------------------------------------------------

各種マスタデータを管理する画面です。6つのサブタブがあります。

[給与テーブル]
等級・号俸ごとの基本給、役職手当、昇給額を管理します。
社員の等級に応じた標準的な給与水準の参考にします。

[法定福利費率]
社会保険の会社負担率を管理します。
- 健康保険（協会けんぽ: 約5%）
- 厚生年金（9.15%）
- 雇用保険（0.95%）
- 労災保険（0.3%）
- 介護保険（0.82%、40歳以上のみ）
- 子ども・子育て拠出金（0.36%）

上限額がある場合は設定でき、月額総支給が上限を超える場合は上限額で計算されます。

[福利厚生費]
通勤手当、住宅手当、家族手当、退職金積立、健康診断、研修費などを管理します。
費用種別（固定/変動/一時）と適用条件（全員/正社員のみ/管理職のみ）を設定できます。

[採用コスト]
求人広告費、エージェント手数料、面接費用、入社準備費、研修費、PC・備品を管理します。
採用シミュレーション時にこのデータが自動で参照されます。

[アラート]
予算超過や人件費率超過などの警告条件を設定します。
条件値（%）を設定すると、月次チェック時に閾値を超えた場合にアラートが発報されます。

[システム]
- シート初期化: 全18シートを作成し、サンプルデータを投入
- 全データ再計算: 全社員の人件費を再計算し、サマリーを再生成
- トリガー設定: 毎月1日の月次集計、毎月15日の予算チェックの自動トリガーを設定


----------------------------------------------------
12. 自動処理（トリガー）
----------------------------------------------------

「設定」>「システム」>「トリガー設定」ボタンで以下の自動処理を設定できます。

- 毎月1日 6:00: 月次集計
  全社員の人件費を再計算し、部署別・職種別・等級別のサマリーを更新します。

- 毎月15日 9:00: 予算チェック
  当月の人件費が予算に対して閾値を超えていないかチェックし、
  超過している場合はアラートメールを送信します。
  また、予算管理シートの実績値を更新します。


----------------------------------------------------
13. スプレッドシートの構成
----------------------------------------------------

本ツールは以下の18シートで構成されています。

1. 社員マスタ: 社員の基本情報と給与データ
2. 給与テーブルマスタ: 等級別の給与体系
3. 法定福利費率マスタ: 社会保険料率
4. 福利厚生費マスタ: 福利厚生費の項目と単価
5. 採用コストマスタ: 採用時に発生するコスト
6. 個人別人件費明細: 自動計算された個人別コスト内訳
7. 採用シミュレーション入力: 採用シミュレーションの条件
8. 採用シミュレーション結果: 採用シミュレーションの算出結果
9. 昇給シミュレーション: 昇給シミュレーションの条件と結果
10. 部署別人件費サマリー: 部署ごとの集計データ
11. 職種別人件費サマリー: 職種ごとの集計データ
12. 等級別人件費サマリー: 等級ごとの集計データ
13. 年間人件費予算管理: 月次の予算と実績
14. 中長期人件費予測: 年度別の予測データ
15. コスト構造分析: 人件費の構成比分析
16. アラート設定: 警告条件の設定
17. シミュレーション履歴: 過去のシミュレーション記録
18. ダッシュボード: サマリーデータ


----------------------------------------------------
14. よくある操作の流れ
----------------------------------------------------

[新入社員を登録する]
1. 「社員管理」を開く
2. 「社員追加」をクリック
3. 必要項目を入力して保存
4. 「人件費明細」を開いて「人件費再計算」をクリック

[新規採用の総コストを確認する]
1. 「採用シミュレーション」を開く
2. 想定する条件（給与、雇用形態、採用人数等）を入力
3. 「シミュレーション実行」をクリック
4. 初年度総コストとコスト倍率を確認

[昇給の影響を確認する]
1. 「昇給シミュレーション」を開く
2. 対象範囲と昇給率を入力
3. 「シミュレーション実行」をクリック
4. 年間の総増加額を確認

[月次の分析を行う]
1. ダッシュボードで「再計算」をクリック（人件費再計算+サマリー再生成）
2. 「分析」タブで各切り口の詳細を確認
3. 「予算管理」で予算との乖離を確認

[マスタデータを変更する]
1. 「設定」を開く
2. 対象のタブを選択（法定福利費率、福利厚生費等）
3. 「編集」ボタンで値を変更
4. 変更後は「人件費明細」の「人件費再計算」で反映
