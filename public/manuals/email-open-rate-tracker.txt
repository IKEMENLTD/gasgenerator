================================================================================
営業メール開封率トラッカー 使い方説明書
================================================================================

■ このシステムでできること
────────────────────────────────────────
・送信した営業メールの記録（送信先・件名・テンプレート等）を一元管理
・メールの開封・返信状況を自動追跡し、開封率・返信率を可視化
・フォロー推奨日を自動算出し、適切なタイミングでのフォローを支援
・スタッフ別・テンプレート別の成績をグラフで比較分析
・月次レポートを管理者へ自動メール配信

■ 画面構成（6タブ）
────────────────────────────────────────
1. 送信ログ        … メール送信記録の登録・閲覧・フォロー管理
2. スタッフ管理    … 営業スタッフの登録・編集・削除
3. 送信先管理      … 送信先（顧客）情報の登録・編集・削除
4. テンプレート管理 … メールテンプレートの登録・編集・削除
5. スタッフ別レポート    … スタッフごとの開封率・返信率・送信件数をグラフ表示
6. テンプレート別レポート … テンプレートごとの開封率・返信率ランキング

================================================================================
■ 各タブの使い方
================================================================================

────────────────────────────────────────
□ 1. 送信ログ
────────────────────────────────────────

【新規登録】
1. 「+ 新規登録」ボタンをクリック
2. 以下の項目を入力：
   ・スタッフ（必須）… ドロップダウンから選択
   ・送信先（必須）… ドロップダウンから選択（会社名 - 担当者名で表示）
   ・テンプレート … 使用したテンプレートがあれば選択（任意）
   ・件名（必須）… 実際に送信したメールの件名
   ・送信日時 … メールを送信した日時（デフォルトは現在日時）
   ・送信ステータス … 送信済 / 失敗 / バウンス
   ・備考 … メモがあれば入力（任意）
3. 「保存」ボタンをクリック

※ トラッキングID・開封フラグ・返信フラグ・開封回数・開封までの時間・
  フォロー推奨日は自動で管理されるため手動入力は不要です。

【フォロー更新】
1. 対象の行の「フォロー更新」ボタンをクリック
2. ダイアログから「フォロー済」または「対応不要」を選択
3. ステータスが即時更新されます

【フィルタ・検索】
・スタッフ … 特定のスタッフの送信ログのみ表示
・送信ステータス … 送信済 / 失敗 / バウンス で絞り込み
・フォロー … 未対応 / フォロー済 / 対応不要 で絞り込み
・検索 … 件名・スタッフ名・送信先名でキーワード検索

【行の色分け（条件付き書式）】
・赤背景 … 未開封かつフォロー推奨日を過ぎている（要フォロー）
・黄背景 … 開封済みだが返信なし（返信待ち）
・緑背景 … 返信あり（対応完了）
・送信ステータス「失敗」… 赤色で強調表示
・送信ステータス「バウンス」… オレンジ色で強調表示

────────────────────────────────────────
□ 2. スタッフ管理
────────────────────────────────────────

【新規登録】
1. 「+ 新規登録」ボタンをクリック
2. 以下の項目を入力：
   ・氏名（必須）
   ・メールアドレス（必須）
   ・部署 … 営業部 / マーケ部 / カスタマーサクセス / 経営企画 / その他
   ・役職 … 担当 / リーダー / マネージャー / 部長 / その他
   ・有効フラグ … 有効 / 無効
3. 「保存」ボタンをクリック

【編集・削除】
・「編集」ボタンで既存データを修正
・「削除」ボタンで確認ダイアログ表示後に削除

【フィルタ】
・部署フィルタ、有効/無効フィルタ、キーワード検索が利用可能

────────────────────────────────────────
□ 3. 送信先管理
────────────────────────────────────────

【新規登録】
1. 「+ 新規登録」ボタンをクリック
2. 以下の項目を入力：
   ・会社名
   ・担当者名
   ・メールアドレス（必須）
   ・業種 … 製造業 / IT / 金融 / 不動産 / 医療 / 小売 / その他
   ・企業規模 … 大企業 / 中堅企業 / 中小企業 / スタートアップ
   ・営業フェーズ … 未接触 / 初回接触 / 商談中 / 提案済 / クローズ / 失注
   ・担当スタッフ … ドロップダウンから選択
   ・配信停止フラグ … 配信可 / 配信停止
   ・備考（任意）
3. 「保存」ボタンをクリック

【フィルタ】
・業種フィルタ、営業フェーズフィルタ、キーワード検索が利用可能

────────────────────────────────────────
□ 4. テンプレート管理
────────────────────────────────────────

【新規登録】
1. 「+ 新規登録」ボタンをクリック
2. 以下の項目を入力：
   ・テンプレート名（必須）
   ・用途カテゴリ … 初回接触 / フォローアップ / 提案 / クロージング / 再アプローチ / その他
   ・件名（必須）… {{担当者名}} などのプレースホルダも使用可能
   ・本文 … メール本文（改行可能）
   ・推奨送信タイミング … 初回接触時 / 開封後1日 / 開封後3日 / 未開封3日後 / 未開封7日後 / 返信後
   ・有効フラグ … 有効 / 無効
3. 「保存」ボタンをクリック

※ 本文はテーブルには表示されません。編集画面で確認・修正できます。
※ 件名・本文をスプレッドシートから直接編集すると、最終更新日が自動更新されます。

────────────────────────────────────────
□ 5. スタッフ別レポート
────────────────────────────────────────

スタッフごとの営業メール成績を確認できます。

【テーブル表示項目】
・集計期間（月）、氏名、部署
・送信件数、成功送信件数、開封件数、開封率
・返信件数、返信率
・平均開封所要時間（時間）
・フォロー実施率
・最終集計日時

【色分け】
・開封率 40%以上 → 緑色（良好）
・開封率 20%未満 → 赤色（要改善）
・返信率 15%以上 → 緑色（良好）
・返信率 5%未満 → 赤色（要改善）

【グラフ（3種類）】
・スタッフ別開封率（月次比較）… グループ縦棒グラフ
・スタッフ別返信率（月次比較）… グループ縦棒グラフ
・スタッフ別送信件数（月次推移）… 折れ線グラフ

【集計実行】
・「集計実行」ボタンで手動集計が可能（通常は毎日08:00に自動実行）

【月フィルタ】
・右上の月選択で、特定の月の集計結果を表示

────────────────────────────────────────
□ 6. テンプレート別レポート
────────────────────────────────────────

テンプレートごとの成績ランキングを確認できます。

【テーブル表示項目】
・集計期間（月）、テンプレート名、用途カテゴリ
・使用回数、開封件数、開封率
・返信件数、返信率
・平均開封所要時間（時間）
・推奨フラグ
・最終集計日時

【色分け】
・推奨フラグ「推奨」 → 行全体が薄緑背景（開封率30%以上かつ返信率10%以上）
・開封率 30%以上 → 緑色
・使用回数 5件未満 → グレー（統計的信頼性が低い）

【グラフ（2種類）】
・テンプレート別開封率ランキング … 横棒グラフ（降順）
・テンプレート別返信率ランキング … 横棒グラフ（降順）
  ※ 使用回数5件未満のテンプレートは薄色で表示

【フォロー件数カード】
・スタッフ別のフォロー未対応件数をカード形式で表示

【集計実行】
・「集計実行」ボタンで手動集計が可能（通常は毎日08:30に自動実行）

================================================================================
■ 自動処理（トリガー）
================================================================================

以下の処理が自動で実行されます：

・毎日 07:00 … 開封・返信ステータスの自動更新
  （トラッキングピクセルのアクセスログ解析 + Gmail返信チェック）
・毎日 07:30 … フォロー推奨日の自動算出 + 未対応フォローのアラートメール送信
・毎日 08:00 … スタッフ別集計の自動更新
・毎日 08:30 … テンプレート別集計の自動更新
・毎月1日 10:00 … 月次レポートを部署管理者（マネージャー・部長）にメール配信

================================================================================
■ フォローアラート機能
================================================================================

・アプリ起動時に、フォロー推奨日が当日以前かつ未対応のメールがある場合、
  画面上部に赤いバナーで件数を表示します。
・バナーをクリックすると、送信ログタブに自動遷移し、
  未対応フォローのみ表示するフィルタが適用されます。
・毎朝、該当スタッフにアラートメールも自動送信されます。

================================================================================
■ フォロー推奨日の算出ルール
================================================================================

・返信済み → フォロー不要（推奨日なし）
・開封済み＋未返信 → 開封日の翌日がフォロー推奨日
・未開封＋送信から3日未満 → 送信日から3日後がフォロー推奨日
・未開封＋送信から3〜6日 → 3日後の推奨日を維持
・未開封＋送信から7日以上 → 送信日から7日後に推奨日を更新

================================================================================
■ 月次レポート
================================================================================

毎月1日の10:00に、前月の成績レポートがメール配信されます。

・送信先: 役職が「マネージャー」または「部長」のスタッフ
・内容: スタッフ別・テンプレート別の開封率・返信率一覧
・ハイライト: チーム平均の80%を下回るスタッフ・テンプレートを要改善として表示

================================================================================
■ 注意事項
================================================================================

・開封追跡はトラッキングピクセル方式のため、メールクライアントの
  画像表示設定によっては検知できない場合があります。
・返信チェックはGmail APIを使用するため、Gmail以外のメールサービスでは
  返信の自動検知ができません。
・集計データは日次で自動更新されますが、リアルタイムではありません。
  最新データが必要な場合は各レポートタブの「集計実行」ボタンをご利用ください。
・テンプレートの推奨フラグは、開封率30%以上かつ返信率10%以上で「推奨」と判定されます。
  使用回数5件未満のテンプレートは統計的信頼性が低いため、グレー表示されます。
