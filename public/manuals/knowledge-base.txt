=====================================
ナレッジベース 使い方説明書
=====================================

■ このツールでできること
・業務ノウハウ・知識を登録・分類・管理
・キーワード・タグ・カテゴリで全文検索
・ナレッジの評価（5段階）・フィードバック
・社内エキスパート（専門家）の登録・検索
・質問・相談の投稿と回答
・ナレッジ活用状況の可視化・分析
・要更新ナレッジの自動検出

■ こんな課題を解決します
・「あの資料どこ？」「誰に聞けばいい？」で時間が浪費される
・業務ノウハウが個人のメモや頭の中に散在している
・退職・異動で知識が失われ、同じ失敗を繰り返す

=====================================
■ 初回セットアップ
=====================================

1. ウェブアプリにアクセスすると初回セットアップ画面が表示されます
2. 氏名・メールアドレス・部署を入力して「利用開始」を押してください
3. 入力した情報はブラウザに保存され、次回以降は自動認識されます
   ※ 右上のユーザー名をクリックすると情報を変更できます

=====================================
■ タブ一覧と使い方
=====================================

【1. ダッシュボード】
アプリを開くと最初に表示される画面です。

・統計カード: 公開ナレッジ数、今月の閲覧数、今月の検索数、平均評価
・人気ナレッジ TOP10: 閲覧数の多いナレッジランキング
・最新ナレッジ: 新しく登録されたナレッジ一覧
・トップエキスパート: 投稿数・回答数の多い専門家
・未解決の質問: 回答待ちの質問一覧
・検索ヒットなしキーワード: 検索しても見つからなかった言葉（ナレッジ不足の手がかり）
・カテゴリ別内訳: ナレッジのカテゴリ分布

【2. ナレッジ検索（検索タブ）】
登録済みナレッジを検索・閲覧する画面です。

〈検索方法〉
・キーワード入力: タイトル、概要、本文、タグ、カテゴリを横断検索
  → タグの同義語も自動でヒットします（例: 「エクセル」で「Excel」も検索）
・フィルタ: カテゴリ、種別、難易度、ステータスで絞り込み
・検索結果をクリックするとナレッジ詳細が表示されます

〈ナレッジ詳細画面〉
・本文の閲覧、添付ファイル・関連リンクの参照
・「いいね」「ブックマーク」ボタン
・「評価する」ボタンで5段階評価+コメント+改善提案を送信
・「編集」ボタンでナレッジの内容を修正

【3. ナレッジ登録（登録タブ）】
新しいナレッジを登録する画面です。

〈入力項目〉
・タイトル（必須）: ナレッジの名前
・概要: 100文字以内の要約
・本文（必須）: 詳細な内容
・カテゴリ（大分類/中分類/小分類）: 管理タブで事前登録したカテゴリから選択
・タグ: カンマ区切りで自由入力（例: Excel,マクロ,自動化）
・ナレッジ種別: 手順書/ノウハウ/事例/Tips/テンプレート/トラブル対応/用語集
・対象者: 全員/特定部署/特定職種
・難易度: 初級/中級/上級
・添付ファイルURL: Google Driveの共有リンク等
・関連ナレッジID / 関連外部リンク
・ステータス: 公開/下書き/レビュー中

〈登録の流れ〉
1. 各項目を入力
2. 「登録」ボタンを押す → ボタンが「登録中...」に変化
3. 登録完了 → チェックマーク付きの完了画面が表示
4. フォームが自動クリアされ、次の登録が可能

【4. エキスパート（エキスパートタブ）】
社内の専門家を登録・検索する画面です。

〈エキスパート登録〉
1. 右上の「+ 登録」ボタンを押す
2. 氏名、部署、役職、専門分野（カンマ区切り）、連絡先、対応可能時間等を入力
3. 「登録」ボタンで保存

〈エキスパート検索〉
・氏名、専門分野、部署で検索可能
・各エキスパートのカードに専門分野タグ、投稿数、回答数、連絡先が表示

【5. 質問・相談（質問タブ）】
ナレッジで解決しない疑問を投稿・回答する画面です。

〈質問を投稿〉
1. 右上の「+ 質問する」ボタンを押す
2. 質問内容と関連カテゴリを入力
3. 「投稿」ボタンで送信 → 関連エキスパートにメール通知
4. 投稿完了画面が表示

〈回答する〉
1. 未回答の質問の「回答する」ボタンを押す
2. 回答内容を入力
3. ステータスを選択（回答済/解決）
4. ナレッジ化候補（はい/いいえ）を選択 → 良い回答はナレッジに昇格可能

〈フィルタ〉
・すべて/未回答/回答済/解決/未解決で絞り込み

【6. 分析（分析タブ）】
ナレッジの活用状況を確認する画面です。

・月次分析サマリー: ナレッジ数、閲覧数、検索回数、ヒット率、評価、質問数
  → 毎月1日に自動集計されます
・部署別状況: 部署ごとのナレッジ登録数、閲覧数、エキスパート数、貢献度
・要更新リスト: 90日以上更新がないナレッジ、低評価ナレッジを自動検出
  → 推奨アクション（更新/統合/廃止）も表示

【7. 管理（管理タブ）】
マスタデータとシステム設定を管理する画面です。

〈カテゴリマスタ〉
・ナレッジのカテゴリ（大分類/中分類/小分類）を登録・管理
・「+ 追加」ボタンで新規カテゴリを追加
  例: 大分類「営業」中分類「提案」小分類「テンプレート」

〈タグマスタ〉
・タグの登録と同義語の設定
・「+ 追加」ボタンで新規タグを追加
  例: タグ名「Excel」同義語「エクセル,スプレッドシート」
  → 同義語を設定すると「エクセル」で検索しても「Excel」タグのナレッジがヒット

〈システム設定〉
・シート初期化: スプレッドシートの全シートにヘッダーを設定（初回のみ）
・トリガー設定: 自動処理（毎日の要更新チェック、毎月の月次分析）を設定

=====================================
■ 自動処理（バックグラウンド）
=====================================

以下の処理が自動で実行されます（管理タブでトリガー設定後）:

・毎日朝7時: 90日以上更新がないナレッジに「要更新」フラグを設定
・毎月1日深夜2時: 月次分析サマリーを自動集計

=====================================
■ 運用のコツ
=====================================

1. まずカテゴリとタグを整備
   → 管理タブでカテゴリ（大/中/小分類）とタグ（同義語付き）を登録
   → 検索精度が大幅に向上します

2. エキスパートを登録
   → 質問投稿時に関連エキスパートに自動通知されます

3. 日常的にナレッジを登録
   → 「教えてもらったこと」「調べてわかったこと」をすぐ登録
   → 下書きステータスで途中保存も可能

4. 評価とフィードバックを活用
   → いいね・ブックマークで良いナレッジを見つけやすく
   → 評価コメント・改善提案でナレッジの品質向上

5. 分析タブを定期確認
   → 検索ヒットなしキーワードからナレッジ不足領域を特定
   → 要更新リストから古いナレッジをメンテナンス

=====================================
■ スプレッドシートのシート構成
=====================================

・ナレッジマスタ: ナレッジ本体（タイトル、本文、カテゴリ、タグ等）
・カテゴリマスタ: カテゴリの大/中/小分類定義
・タグマスタ: タグと同義語の管理
・エキスパートマップ: 専門家の情報と専門分野
・閲覧利用履歴: 誰がどのナレッジを利用したかの記録
・検索ログ: 検索キーワードと結果の記録
・評価フィードバック: 5段階評価とコメント
・更新履歴: ナレッジの変更履歴
・質問相談ログ: Q&Aの記録
・分析サマリー: 月次集計データ
・部署別状況: 部署ごとの貢献度
・要更新リスト: メンテナンス対象ナレッジ
