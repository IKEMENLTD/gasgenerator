  売上ダッシュボード　使い方説明書
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

このシステムは、日々の売上データを入力するだけで、
売上状況の把握・目標管理・分析・予測まで自動で行えるツールです。

PCでもスマホでも使えます。
スマホの場合は、左上の「≡」ボタンでメニューを開いてください。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  毎日やること
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 売上を登録する

  メニューから「売上登録」を開きます。

  1. 売上日を入力します（初期値は今日の日付です）
  2. 担当者を選びます → 部署が自動で入ります
  3. 商品を選びます → 単価と原価が自動で入ります
  4. 顧客を選びます → 業種とランクが自動で入ります
  5. 数量を入力します → 売上金額や粗利が自動計算されます
  6. 種別（新規/既存/リピート）と経路（直販/紹介/Web/展示会）を選びます
  7. 「登録」を押して、確認画面で「OK」を押せば完了です

  ※ 登録したデータはスプレッドシートに自動保存されます。
  ※ 集計やグラフにもすぐ反映されます。


■ ダッシュボードで状況を確認する

  メニューから「ダッシュボード」を開きます。
  開くだけで、以下の情報がまとめて表示されます。

  ・今月の売上・粗利・件数・前年比（画面上部のカード）
  ・目標に対する達成率（プログレスバー）
  ・日別の売上推移グラフ
  ・月別の売上推移グラフ（今年 vs 前年）
  ・担当者別ランキング
  ・商品カテゴリ別の売上構成（円グラフ）
  ・新規 vs 既存の比率
  ・粗利率の推移
  ・顧客別売上ランキング
  ・アラート通知
  ・月末の着地見込み予測


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  売上データを確認・検索する
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  メニューから「売上データ一覧」を開きます。

  画面上部に、件数・売上合計・粗利合計・平均単価が表示されます。

  フィルタで絞り込めます：
  ・年月、担当者、商品、顧客、種別を選んで「検索」を押す
  ・「リセット」を押すと条件がクリアされます

  テーブルは新しい順に7件表示されます。
  「全て表示」を押すと残りのデータも見られます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  目標を設定・管理する
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  メニューから「売上目標管理」を開きます。

■ 目標を確認する

  年度・月・対象のフィルタで絞り込めます。
  テーブルには目標額と実績・達成率が並んで表示されます。
  新しい順に10件表示されます。

  達成率は色で一目でわかります：
  ・緑 = 100%以上　・青 = 80%以上　・黄 = 50%以上　・赤 = 50%未満

■ 目標を登録する

  1. 右上の「新規追加」ボタンを押します
  2. 年度・月・対象（全社/部署/担当者）・金額などを入力します
  3. 「保存」を押せば完了です

  ★ 重要：ダッシュボードの達成率を表示するには、
    毎月「全社」の目標を登録してください。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  分析を見る
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  メニューの「分析」カテゴリに3つの画面があります。
  どの画面も、上部の年月を変えるだけで対象月を切り替えられます。

■ 担当者別分析
  誰がどれくらい売り上げているか、目標に対してどうかを確認できます。
  ・担当者数、売上合計、平均売上、最高売上のカード
  ・担当者別売上のグラフと達成率のグラフ
  ・順位・売上・達成率・粗利・前月比のテーブル

■ 商品別分析
  どの商品が売れているか、構成比はどうかを確認できます。
  ・商品数、売上合計、販売数量、平均粗利率のカード
  ・売上構成の円グラフと売上金額の棒グラフ
  ・順位・売上・数量・粗利率・構成比・前月比のテーブル

■ 顧客別分析
  どの顧客からの売上が多いか、動向はどうかを確認できます。
  ・顧客数、売上合計、取引件数、最高売上のカード
  ・順位・売上・粗利・件数・累計売上のテーブル


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  サマリーを見る
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 日次サマリー
  メニューから「日次サマリー」を開きます。
  年月を選ぶと、その月の日ごとの売上集計が表示されます。
  新しい日付が上に来ます（10件表示）。

  各日の売上金額・粗利・前日比・前年同日比・月累計・進捗率が
  一覧で確認できます。
  前日比・前年比は ▲（緑=上昇）▼（赤=下降）で表示されます。

■ 月次サマリー
  メニューから「月次サマリー」を開きます。
  月ごとの売上目標・実績・達成率・粗利・前月比・前年同月比が
  一覧で確認できます（10件表示、新しい月が上）。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  アラートを使う
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  メニューから「アラート管理」を開きます。

  あらかじめ4つのアラートが設定されています：
  ・月中進捗率アラート … 15日時点で達成率50%未満なら通知
  ・粗利率低下アラート … 粗利率20%未満なら毎日通知
  ・前年比大幅減アラート … 前年比80%未満なら週次で通知
  ・日次売上件数アラート … 営業日に売上0件なら毎日通知

  通常は毎日夕方17時に自動チェックされます。
  今すぐチェックしたい場合は、画面上部の
  「アラートチェック実行」ボタンを押してください。

  検出されたアラートは画面下部の「アラートログ」に記録されます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  売上予測を見る
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  メニューから「売上予測」を開きます。
  毎朝自動で更新される予測データが表示されます。

  ・現時点の実績と、このペースで行くと月末いくらになるかの予測
  ・目標に対する達成見込み（緑=達成見込み、赤=達成困難）
  ・楽観/標準/悲観の3つのシナリオ別予測
  ・残り営業日、目標達成に必要な1日あたりの売上額
  ・前年同月の実績との比較


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  マスタデータの管理
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  担当者・商品・顧客の情報は、メニューの「マスタ管理」から
  一覧で確認できます（それぞれ10件表示、新しい順）。

  データの追加や変更は、Googleスプレッドシートを直接開いて
  該当のシートを編集してください。

  ・担当者を追加 → 「担当者マスタ」シートに行を追加（ID例: STF-009）
  ・商品を追加 → 「商品マスタ」シートに行を追加（ID例: PRD-011）
  ・顧客を追加 → 「顧客マスタ」シートに行を追加（ID例: CUS-016）

  ※ 1行目（ヘッダー行）は絶対に削除しないでください。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  自動で動いている処理
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  以下の処理は自動で動くため、手動操作は不要です。

  ・スプレッドシートが編集されると → サマリーが即時更新
  ・毎日 朝7時 → 全サマリーと売上予測を更新
  ・毎日 夕方17時 → アラート条件をチェック
  ・毎週月曜 → 週次レポートを生成
  ・毎月1日 → 月次レポートを生成

  困ったときは
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ・画面が真っ白、データが出ない
    → ブラウザをリロード（PCならCtrl+F5）してください

  ・データが古いまま更新されない
    → 通常は毎朝7時に自動更新されます
    → すぐ反映したい場合は管理者にご連絡ください

  ・グラフが表示されない
    → その月に売上データがない場合、グラフは表示されません

  ・ダッシュボードの達成率が出ない
    → 当月の「全社」目標が登録されているか確認してください

  ・スマホでメニューが出ない
    → 画面左上の「≡」ボタンをタップしてください
