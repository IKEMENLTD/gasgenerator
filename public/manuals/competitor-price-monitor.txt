==================================================
  競合価格モニタリング 使い方説明書
==================================================

■ このシステムでできること
--------------------------------------------------
競合商品の価格変動を自動で監視し、変動を検出したら
メールやSlackで即座に通知するシステムです。

・自社商品と競合商品の価格を一元管理
・価格変動を自動検出してアラート通知
・競合との価格差を横並びで比較
・価格推移グラフで値動きを可視化
・月次レポートで価格戦略の振り返りを支援

EC・小売・SaaS企業の価格担当者・マーケティング担当者
に最適です。


■ 画面構成（5つのタブ）
--------------------------------------------------

【タブ1】ダッシュボード（トップ画面）
【タブ2】商品管理
【タブ3】価格履歴
【タブ4】アラート履歴
【タブ5】競合比較


==================================================
■ タブ1: ダッシュボード
==================================================

アプリを開くと最初に表示される画面です。
6つの指標・グラフで価格モニタリングの状況を一覧できます。

● 監視商品数（KPIカード）
  現在有効になっている監視対象商品の数を表示します。

● 本日のアラート発生件数（KPIカード）
  今日送信されたアラートの件数を表示します。
  件数が1以上の場合は警告色で表示されます。

● 価格変動が多い競合TOP5（横棒グラフ）
  直近30日間で価格変動が多かった競合会社を
  上位5社まで棒グラフで表示します。

● 自社vs競合 価格差率の分布（円グラフ）
  競合比較データから「自社が安い」「同額」「競合が安い」
  の構成比率を円グラフで表示します。
  ツールチップにはパーセンテージが表示されます。

● 特定商品の価格推移（折れ線グラフ）
  ドロップダウンで商品を選択すると、
  直近30日間の価格推移が折れ線グラフで表示されます。
  Y軸は円単位で表示されます。

● アラート送信成功率（ゲージチャート）
  直近7日間のアラート送信成功率をゲージで表示します。
  100%が理想で、失敗があると割合が下がります。


==================================================
■ タブ2: 商品管理
==================================================

監視対象の商品（自社・競合）を登録・編集・削除する画面です。

--------------------------------------------------
● 商品の新規追加
--------------------------------------------------
1. 「+ 新規追加」ボタンをクリック
2. フォームに以下を入力:
   - 商品名（必須）
   - カテゴリ（EC / 小売 / SaaS から選択）
   - 自社/競合（自社 または 競合 を選択）
   - 競合会社名（競合の場合に入力）
   - 監視対象URL（商品ページのURL）
   - 自社対応商品ID（競合の場合、比較する自社商品を選択）
   - 基準価格（監視開始時点の価格）
   - 閾値%（この変動率を超えたらアラート。デフォルト5%）
   - 通知先メール（アラート送信先）
   - 有効/無効（監視を続けるかどうか）
   - 備考
3. 「保存」をクリック
   → 商品IDが自動採番されます（P001, P002...）

--------------------------------------------------
● 商品の編集
--------------------------------------------------
テーブルの「編集」ボタンをクリックすると
フォームに既存データがセットされます。
変更して「保存」をクリックしてください。

--------------------------------------------------
● 商品の削除
--------------------------------------------------
テーブルの「削除」ボタンをクリックすると
確認ダイアログが表示されます。
「実行」をクリックすると削除されます。

--------------------------------------------------
● 有効/無効の切替
--------------------------------------------------
商品の「編集」から有効/無効を変更できます。
無効にすると価格収集の対象から除外されます。

--------------------------------------------------
● フィルタ・検索
--------------------------------------------------
カードヘッダー内のドロップダウン・検索欄で
カテゴリ、自社/競合、有効/無効、キーワードで
絞り込みができます。

--------------------------------------------------
● 価格手動登録
--------------------------------------------------
1. 「価格手動登録」ボタンをクリック
2. 商品をドロップダウンから選択
3. 取得価格を入力
4. 「登録」をクリック
   → 価格履歴に記録されます
   → 前回価格との変動率が自動計算されます
   → 閾値を超えた場合は変動フラグが立ちます


==================================================
■ タブ3: 価格履歴
==================================================

収集された価格データの履歴を確認する画面です。
データは新しい順に表示されます。

--------------------------------------------------
● 表示項目
--------------------------------------------------
ログID / 商品ID / 商品名 / 記録日時 / 取得価格 /
前回価格 / 変動額 / 変動率(%) / 変動フラグ /
送信済 / 取得方法

--------------------------------------------------
● 条件付き表示
--------------------------------------------------
・変動フラグが○の行: 薄赤背景・赤太字で強調表示
・変動額が正（値上がり）: 赤文字
・変動額が負（値下がり）: 青文字
・初回登録（前回価格なし）: 前回価格・変動額・変動率は「-」

--------------------------------------------------
● フィルタ
--------------------------------------------------
・商品ドロップダウン: 特定の商品だけ表示
・開始/終了日: 期間で絞り込み
・検索: キーワードで絞り込み

--------------------------------------------------
● CSV出力
--------------------------------------------------
「CSV出力」ボタンをクリックすると、
現在のフィルタ条件に該当するデータがCSVで
ダウンロードされます。Excelで文字化けしません。


==================================================
■ タブ4: アラート履歴
==================================================

送信されたアラートの履歴を確認する画面です。
データは新しい順に表示されます。

--------------------------------------------------
● 表示項目
--------------------------------------------------
アラートID / 送信日時 / 商品名 / 変動前価格 /
変動後価格 / 変動率 / 通知先 / チャネル /
ステータス / 操作

--------------------------------------------------
● 条件付き表示
--------------------------------------------------
・ステータス「成功」: 青バッジ
・ステータス「失敗」: 行全体がオレンジ背景・太字
・ステータス「再送待ち」: 黄バッジ

--------------------------------------------------
● アラートの再送
--------------------------------------------------
ステータスが「失敗」のアラートのみ、
「再送」ボタンが表示されます。
クリックするとアラートが再送信されます。

--------------------------------------------------
● フィルタ
--------------------------------------------------
ステータス（全て/成功/失敗/再送待ち）と
キーワードで絞り込みができます。


==================================================
■ タブ5: 競合比較
==================================================

自社商品と競合商品の最新価格を横並びで比較する画面です。

--------------------------------------------------
● 表示項目
--------------------------------------------------
自社商品名 / 自社最新価格 / 競合会社名 /
競合最新価格 / 価格差 / 価格差率(%) /
価格優位性 / 最終更新

--------------------------------------------------
● 条件付き表示
--------------------------------------------------
・「競合が安い」: 薄赤背景・太字で警告表示
・「自社が安い」: 薄緑背景で安心表示

--------------------------------------------------
● 比較データ更新
--------------------------------------------------
「比較データ更新」ボタンをクリックすると、
最新の価格データで比較表を再計算します。
通常は毎日自動更新されます。

--------------------------------------------------
● ソート
--------------------------------------------------
「自社商品名」「価格差」「価格差率(%)」の列ヘッダーを
クリックすると昇順/降順でソートできます。


==================================================
■ 自動処理（バックグラウンド）
==================================================

以下の処理が自動で実行されます。
設定変更はスプレッドシートの「設定」シートで行います。

● 価格収集（6時間ごと）
  有効な商品のURLから価格を自動取得します。
  取得できない場合は基準価格を使用します。

● アラート送信（価格収集の直後）
  価格変動が閾値を超えた場合に、
  メールまたはSlackで通知します。
  同一商品への連続通知は24時間抑制されます。

● 競合比較更新（毎日朝）
  自社と競合の最新価格を比較して
  競合比較シートを更新します。

● ダッシュボード更新（毎日朝）
  各指標を再計算してダッシュボードを更新します。

● 月次レポート送信（毎月1日）
  前月の価格変動回数・アラート件数・
  競合会社別変動ランキングをまとめた
  PDFレポートを管理者メールに送信します。


==================================================
■ 設定項目（スプレッドシートの「設定」シート）
==================================================

以下の設定はスプレッドシートの「設定」シートで
直接編集してください。

・SCRAPING_INTERVAL_HOURS: 価格収集の間隔（時間）
  初期値: 6

・SLACK_WEBHOOK_URL: Slack通知用のWebhook URL
  初期値: 空欄（設定するとSlack通知が有効になります）

・DEFAULT_ALERT_THRESHOLD_PCT: デフォルトのアラート閾値(%)
  初期値: 5

・ADMIN_EMAIL: 管理者メールアドレス
  初期値: 空欄（月次レポートの送信先）

・MAX_RETRY_COUNT: 価格取得失敗時の最大リトライ回数
  初期値: 3

・ALERT_COOLDOWN_HOURS: 同一商品の連続アラート抑制時間（時間）
  初期値: 24


==================================================
■ 活用シーン
==================================================

・EC事業者が競合ショップの価格を毎日チェックしたい
・SaaS企業が競合サービスのプラン料金改定をすぐに把握したい
・小売業で特売価格や値下げのタイミングを逃さず対応したい
・価格戦略会議に最新の競合価格データを持ち込みたい
・月次で価格動向を振り返り、次の戦略に活かしたい


==================================================
■ モバイル対応
==================================================

スマートフォン・タブレットからもアクセスできます。
画面左上のハンバーガーメニュー（三本線）をタップすると
サイドバーが開き、各タブに移動できます。
テーブルは横スクロールで全列を確認できます。
