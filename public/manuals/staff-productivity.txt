============================================
スタッフ別生産性システム 使い方説明書
============================================

このシステムは、スタッフごとの売上・生産性を自動集計し、
目標達成率やランキング、成長推移を可視化するツールです。
PC・スマートフォン両対応です。

============================================
【基本操作】
============================================

■ 画面の切り替え
  - 左のサイドバーから表示したい画面をクリック
  - ≡ボタン（ハンバーガーメニュー）でサイドバーの表示/非表示を切り替え
  - スマホ: ≡ボタンをタップ → メニュー選択 → 自動で閉じる

■ フィルター（絞り込み）
  - 各画面の上部にある「年月」「店舗」「スタッフ」の
    プルダウンで表示データを絞り込みできます

■ テーブル（一覧表）
  - 1画面に最大15件表示、それ以上はスクロールで確認
  - 列が多い場合は横スクロールで確認
  - テーブルのヘッダー行はスクロールしても固定表示

■ 分析の手動実行
  - 右上の「分析実行」ボタンをクリック
  - 当月のデータを再集計・分析します
  - 自動実行もされますが、すぐに結果を見たい場合に使用

============================================
【画面ごとの使い方】（全15画面）
============================================

────────────────────────────
1. ダッシュボード
────────────────────────────
  システムの全体像を把握する画面です。

  ● 上部KPIカード（5枚）
    - 今月の総売上 / 売上目標 / 全体達成率 / 平均生産性 / スタッフ数
    - 前月比の増減も表示

  ● 売上ランキングTOP10
    - 今月の売上上位10名を表示

  ● 時間あたり生産性ランキング
    - 1時間あたりの売上が高いスタッフ順に表示

  ● 達成率分布グラフ（円グラフ）
    - 120%以上 / 100-120% / 80-100% / 60-80% / 60%未満 の人数分布

  ● 店舗別平均生産性比較（棒グラフ）
    - 各店舗の平均売上を比較

  ● 成長著しいスタッフTOP5
    - 前月比で大きく成長したスタッフを表示
    ※ 2ヶ月以上のデータが必要です

  ● 要教育スタッフアラート
    - 要教育・要注意に該当するスタッフ一覧

  ● 目標未達アラート
    - 「やや遅れ」「要対策」のスタッフ一覧

  ● スタッフ別前月比（棒グラフ）
    - 各スタッフの前月からの売上増減

  ● 優秀者表彰候補
    - 達成率110%以上かつ生産性TOP3のスタッフ

────────────────────────────
2. スタッフマスタ
────────────────────────────
  スタッフの基本情報を管理する画面です。

  ● スタッフ追加
    1.「スタッフ追加」ボタンをクリック
    2. 入力フォームに以下を入力:
       氏名 / カナ / 所属店舗 / 雇用形態 / 役職 /
       入社日 / 時給・月給 / 資格・スキル / 担当メニュー /
       月間目標売上 / 月間目標件数 / メールアドレス
    3.「登録」ボタンで保存
    ※ IDは自動採番されます（STF-009, STF-010, ...）

  ● スタッフ編集
    1. 一覧の「編集」ボタンをクリック
    2. 変更したい項目を修正
    3.「更新」ボタンで保存

  ● スタッフ削除（退職処理）
    - 編集画面でステータスを「退職」に変更

────────────────────────────
3. 勤怠データ
────────────────────────────
  スタッフの出退勤情報を管理する画面です。

  ● 勤怠登録
    1.「勤怠登録」ボタンをクリック
    2. 入力フォームに以下を入力:
       日付 / スタッフ / 出勤時刻 / 退勤時刻 / 休憩時間 / 区分
    3.「登録」ボタンで保存
    ※ 実働時間は自動計算されます

  ● フィルター
    - 年月: 表示する月を選択
    - 店舗: 「全店舗」または特定の店舗で絞り込み

────────────────────────────
4. 売上データ
────────────────────────────
  施術・販売ごとの売上情報を管理する画面です。

  ● KPIカード（4枚）
    - 売上合計 / 粗利合計 / 件数 / 平均単価

  ● 売上登録
    1.「売上登録」ボタンをクリック
    2. 入力フォームに以下を入力:
       売上日時 / スタッフ / 顧客名 / 顧客種別 /
       メニュー名 / カテゴリ / 単価 / 数量 / 原価 /
       担当区分 / オプション売上 / 店販売上 / 備考
    3.「登録」ボタンで保存
    ※ 売上金額（単価×数量）と粗利（売上-原価）は自動計算

  ● フィルター
    - 年月 / 店舗で絞り込み

────────────────────────────
5. 日次サマリー
────────────────────────────
  スタッフごとの1日の実績をまとめた画面です。
  ※ データは自動生成されます（手動入力不要）

  表示項目:
  - 日付 / スタッフ名 / 店舗 / 実働時間 / 件数 /
    売上金額 / 粗利 / 時間あたり売上 / 件数あたり単価

  ● フィルター
    - 年月 / スタッフ / 店舗で絞り込み

────────────────────────────
6. 月次サマリー
────────────────────────────
  スタッフごとの月間実績をまとめた画面です。
  ※ データは自動生成されます

  ● KPIカード（4枚）
    - 総売上 / 総目標 / 平均達成率 / 平均時間あたり売上

  表示項目:
  - スタッフ名 / 店舗 / 売上金額 / 売上目標 / 達成率（プログレスバー付き）/
    時間あたり売上 / 前月比（▲緑/▼赤）/ 前年同月比 / 評価ランク

  ● フィルター
    - 年月 / スタッフ / 店舗で絞り込み

────────────────────────────
7. 時間帯別生産性
────────────────────────────
  時間帯ごとの売上パフォーマンスを分析する画面です。

  時間帯区分: 9-12時 / 12-15時 / 15-18時 / 18-21時

  表示項目:
  - スタッフ名 / 各時間帯の売上・件数 / 得意時間帯（◎マーク）

  ● グラフ
    - 時間帯別売上分布（棒グラフ）

  ● フィルター
    - 年月 / スタッフで絞り込み

────────────────────────────
8. メニュー別実績
────────────────────────────
  メニューカテゴリごとの売上実績を分析する画面です。

  表示項目:
  - カテゴリ名 / 件数 / 売上金額 / 構成比 / 得意メニュー（◎マーク）

  ● グラフ
    - カテゴリ別売上構成（ドーナツグラフ）

  ● フィルター
    - 年月 / スタッフで絞り込み

────────────────────────────
9. 成長推移分析
────────────────────────────
  スタッフの売上成長を月ごとに追跡する画面です。

  表示項目:
  - 年月 / 売上金額 / 達成率 / 時間あたり売上 /
    3ヶ月移動平均 / 成長率（前月比）/ 成長トレンド（上昇/横ばい/下降）

  ● グラフ
    - 売上成長推移（折れ線グラフ）

  ● フィルター
    - スタッフで絞り込み

  ※ 2ヶ月以上のデータがないと成長率・トレンドは表示されません

────────────────────────────
10. 生産性ランキング
────────────────────────────
  スタッフの売上ランキングを表示する画面です。

  ● TOP3カード
    - 1位（金）/ 2位（銀）/ 3位（銅）のスタッフをカード表示

  ● ランキングテーブル
    - 順位 / スタッフ名 / 店舗 / 売上金額 / 達成率 /
      時間あたり売上 / 評価ランク / 順位変動（↑↓）

  ● 評価ランク基準
    - S: 達成率120%以上 かつ 生産性130%以上
    - A: 達成率100%以上 かつ 生産性100%以上
    - B: 達成率80%以上 かつ 生産性80%以上
    - C: 達成率60%以上 かつ 生産性60%以上
    - D: 上記以外

  ● フィルター
    - 店舗で絞り込み

────────────────────────────
11. 教育対象・優秀者
────────────────────────────
  スタッフを実績に基づいて自動分類する画面です。

  ● 分類別カウントカード（4枚）
    - 優秀者 / 標準 / 要教育 / 要注意 の人数

  ● 分類基準
    - 優秀者: 達成率110%以上 かつ 生産性TOP3
    - 標準:   達成率80%以上
    - 要教育: 達成率60〜80%
    - 要注意: 達成率60%未満

  表示項目:
  - スタッフ名 / 店舗 / 分類 / 判定理由 / 強み / 課題 /
    推奨アクション / 指導者 / 目標 / 更新日

  ● フィルター
    - 店舗 / 分類で絞り込み

────────────────────────────
12. 目標進捗管理
────────────────────────────
  月間目標に対する進捗状況を管理する画面です。

  ● ステータス別カウントカード（3枚）
    - 順調 / やや遅れ / 要対策 の人数

  表示項目:
  - スタッフ名 / 目標種別（売上/件数）/ 目標値 / 実績 /
    達成率（プログレスバー付き）/ ステータス

  ● ステータス基準
    - 順調:   進捗率が月の経過日数割合以上
    - やや遅れ: 進捗率が経過日数割合の80%以上
    - 要対策:  進捗率が経過日数割合の80%未満

  ● フィルター
    - 年月で絞り込み

────────────────────────────
13. 店舗別比較
────────────────────────────
  店舗ごとのパフォーマンスを比較する画面です。

  ● 店舗KPIカード
    - 店舗名 / 総売上 / 平均生産性 / ランク（A/B/C）

  ● 店舗別生産性比較グラフ（棒グラフ）
    - 平均売上/人 と 平均時間あたり売上 を比較

  ● 店舗別スタッフ比較テーブル
    - 店舗名 / スタッフ数 / 総売上 / 平均売上/人 /
      平均時間あたり売上 / 最高生産性 / 最低生産性 / 格差（倍率）/ ランク

  ● フィルター
    - 年月で絞り込み

────────────────────────────
14. アラート管理
────────────────────────────
  自動アラートの設定を管理する画面です。

  ● アラート一覧（4種類）
    - 達成率低下アラート（基準: 70%）
    - 生産性低下アラート（基準: 70%）
    - 順位急落アラート（基準: 5位以上低下）
    - リピート率低下アラート（基準: 30%）

  ● 操作
    -「無効化」ボタン: アラートを停止
    -「有効化」ボタン: アラートを再開

────────────────────────────
15. アラートログ
────────────────────────────
  検出されたアラートの履歴を確認する画面です。

  表示項目:
  - ID / 検出日 / スタッフ名 / 店舗 / アラート種別 /
    検出値 / 基準値 / ステータス

  ● 操作
    -「対応済」ボタン: ステータスを「対応済」に変更

  ● フィルター
    - ステータス（全て/未対応/対応中/対応済）で絞り込み

============================================
【日常の運用フロー】
============================================

■ 毎日の作業
  1. 勤怠データを登録（出退勤時刻・休憩時間）
  2. 売上データを登録（施術・販売ごと）
  ※ 日次サマリーは毎日朝7時に自動生成されます
  ※ すぐ確認したい場合は「分析実行」ボタンを押してください

■ 毎月の確認
  1. ダッシュボードで全体KPIを確認
  2. 月次サマリーで各スタッフの達成率を確認
  3. 生産性ランキングで順位を確認
  4. 教育対象・優秀者で育成対象を確認
  5. 目標進捗管理で遅れているスタッフをフォロー

■ スタッフの入退社時
  1. スタッフマスタで「スタッフ追加」または
     ステータスを「退職」に変更

■ アラート対応
  1. ダッシュボードの「要教育スタッフアラート」を確認
  2. アラートログで詳細を確認
  3. 対応後「対応済」ボタンをクリック

============================================
【自動で行われること】（操作不要）
============================================

  - 毎日朝7時: 前日の日次サマリー＋月次サマリー更新
  - 毎日夕方17時: 目標進捗チェック＋アラート検出
  - 毎週月曜9時: 週次レポートメール配信
  - 毎月1日6時: 前月の全分析＋月次レポートメール配信
  - スプシ編集時: 関連する日次サマリーを自動再生成

============================================
【スマートフォンでの操作】
============================================

  - ヘッダー左の ≡ ボタンをタップ → メニュー表示
  - メニュー項目をタップ → ページ切替＋メニュー自動閉じ
  - テーブルは横スクロール対応
  - 各種登録フォームもスマホ対応済み

============================================
【困ったときは】
============================================

  ● データが最新でない場合
    →「分析実行」ボタンをクリックして手動で再分析

  ● 登録したデータが表示されない場合
    → ページを再読み込み（スマホ: 下に引いてリロード）

  ● グラフが表示されない場合
    → ブラウザを再読み込みしてください

  ● 「読み込み中」のまま動かない場合
    → ブラウザを再読み込みしてください
    → それでも改善しない場合はGASのトリガーが
       正常に動作しているか確認してください
