====================================
目標コミット管理 - 使い方説明書
====================================

------------------------------------
このシステムでできること
------------------------------------

目標コミット管理は、目標の設定から進捗管理、週次レビュー、
達成までを一元管理するWebアプリケーションです。

- 個人目標・チーム目標の登録と進捗管理
- 達成率の自動計算とグラフによる可視化
- 週次レビューによる定期的な振り返り
- 目標達成期限が近い場合の自動アラート通知
- バッジ機能によるモチベーション維持
- チームメンバーの目標進捗の一覧把握
- CSV形式でのレポート出力
- 操作ログ・エラーログの自動記録

個人利用（自己成長・キャリアアップ）から、
チーム利用（2〜50名程度の部署・チーム）まで対応しています。

------------------------------------
画面一覧と操作方法
------------------------------------

[1] ユーザー登録画面
  初回アクセス時に表示されます。
  - 表示名: 自分の名前を入力
  - 所属チーム: チームがあれば選択（後から変更可能）
  - 「登録する」ボタンで登録完了
  ※最初に登録したユーザーは自動的に「管理者」になります。

[2] ダッシュボード
  ログイン後のメイン画面です。
  - 統計カード: 進行中の目標数、達成した目標数、平均達成率、注意が必要な目標数
  - 週次レビュー未提出バナー: 未提出の場合にオレンジ色で表示
  - 進行中の目標一覧: 各目標のプログレスバーと達成率を表示
  - 目標達成率チャート: 棒グラフで各目標の達成率を可視化
  - 注意が必要な目標: 期限7日以内で達成率50%未満の目標を強調表示
  ※目標カードをクリックすると詳細画面が開きます。

[3] 目標管理
  目標の一覧表示、新規登録、編集、達成宣言、キャンセルを行います。

  ●新規目標登録
  - 「新規目標」ボタンをクリック
  - 目標名（必須）、目標詳細、カテゴリ、タイプを入力
  - 目標値（必須）、現在値、単位を入力
  - 開始日、達成期限（必須）を設定
  - 重要度（高/中/低）を選択
  - 個人目標またはチーム目標を選択
  - チーム目標の場合はチームを指定
  - 上位目標があれば選択
  - 「登録」ボタンで保存

  ●目標の編集
  - 一覧の「編集」ボタンをクリック
  - 内容を修正して「更新」ボタンで保存

  ●目標達成宣言
  - 一覧の「達成」ボタンをクリック
  - 最終達成値と達成コメントを入力
  - 「達成を宣言」ボタンで確定
  - 紙吹雪アニメーションでお祝いメッセージが表示されます

  ●検索・フィルタ
  - 検索バーに入力すると目標名で絞り込み
  - ステータス（未着手/進行中/達成/未達成/保留）でフィルタ
  - 重要度（高/中/低）でフィルタ
  - テーブルのヘッダーをクリックでソート

[4] 進捗記録
  目標の進捗を記録します。
  - 対象目標をプルダウンから選択
  - 記録日を設定（初期値は本日）
  - 記録値を入力（現在の数値）
  - 進捗コメントを入力
  - 「進捗を更新」ボタンで保存
  - 達成率が自動計算されて更新されます
  - 目標値に到達した場合は自動的に「達成」になります

  下部に進捗履歴テーブルが表示されます。

[5] 週次レビュー
  毎週の振り返りを記録・提出します。
  - 対象週が自動表示されます
  - 今週の全体的な進捗コメント（必須）
  - 今週の課題点（必須）
  - 来週のアクションプラン（必須）
  - 各目標への個別コメント（任意）
  - 「レビューを提出」ボタンで提出

  下部に過去のレビュー履歴が表示されます。
  未提出の場合はダッシュボードにリマインダーが表示されます。

[6] 履歴・振り返り
  完了した目標（達成・未達成）の一覧を表示します。
  - 目標名、達成率、期限、ステータスを確認
  - 過去の目標を振り返り、次の目標設定に活用できます

[7] レポート
  目標の進捗状況をCSVファイルで出力します。
  - レポート種別: 週次進捗 / 月次目標達成 / 個人パフォーマンス
  - 対象期間を開始日・終了日で指定
  - 「レポートを生成(CSV)」ボタンでCSVファイルを生成
  - ダウンロードリンクが表示されます

[8] バッジ
  獲得したバッジと全バッジ一覧を表示します。
  - 獲得済みバッジ: カラーで表示
  - 未獲得バッジ: グレーアウトで表示
  - バッジの種類:
    - 初目標達成: 最初の目標を達成
    - 週次レビュー皆勤賞: 4週連続でレビュー提出
    - 目標マスター: 5つ以上の目標を達成
    - 進捗の鬼: 30回以上進捗を記録
    - 半期MVP: 半期で最も多くの目標を達成

[9] チーム管理
  チームの作成・管理を行います。
  - 「新規チーム」ボタンでチームを作成
  - チーム名、説明、マネージャーを設定
  - チームの無効化も可能

[10] ユーザー管理
  ユーザーの一覧表示と編集を行います。
  - 表示名、メール、チーム、役割、ステータスを確認
  - 「編集」ボタンで表示名を変更

[11] 設定
  プロフィールとシステム設定を管理します。

  ●プロフィール
  - 表示名の変更
  - 所属チームの変更

  ●システム設定
  - 週次レビューリマインダーの曜日
  - 日次リマインダーの時刻
  - アラート発火までの残り日数閾値
  - アラート発火の達成率閾値
  - アーカイブ対象の月数
  - バックアップ先フォルダID

  ●管理操作
  - シート初期化: スプレッドシートのシートとマスタデータを初期化
  - トリガー設定: 自動実行トリガー（日次リマインダー、週次レビュー等）を設定

  ●テストデータ
  - テストデータ挿入: サンプルデータを一括挿入（チーム3件、ユーザー5件、目標10件、進捗15件、レビュー3件等）
  - 全データクリア: 全データを削除（マスタデータと現在のユーザーは保持）

------------------------------------
自動実行機能（トリガー）
------------------------------------

設定画面で「トリガー設定」を実行すると、以下の自動処理が有効になります。

- 毎日 09:00: 日次進捗チェック＆リマインダーメール送信
- 毎週月曜 09:00: 週次レビュー未提出者にリマインダーメール送信
- 毎週月曜 00:00: 前週の週次レビュー集計
- 毎月1日 00:00: 前月の目標達成状況集計＆月次レポートメール送信
- 毎日 17:00: 目標未達アラート通知
- 毎日 03:00: バッジ付与処理
- 毎月1日 02:00: 古いデータのアーカイブ処理
- 毎週日曜 02:00: スプレッドシート全体の自動バックアップ

------------------------------------
通知機能
------------------------------------

- ヘッダーのベルアイコンに未読通知数が表示されます
- クリックすると通知一覧がドロップダウンで表示されます
- 通知をクリックすると既読になります
- メール通知とアプリ内通知の2種類があります

------------------------------------
モバイル対応
------------------------------------

- 768px以下の画面幅でハンバーガーメニューが表示されます
- 左上のメニューボタンをタップするとサイドバーが開きます
- メニュー項目をタップするとサイドバーが自動的に閉じます
- 全ての画面がレスポンシブ対応しています

------------------------------------
データ構成（スプレッドシート）
------------------------------------

全17シートで構成されています。

マスタデータ:
- USERS_MST: ユーザー情報
- TEAMS_MST: チーム情報
- TARGET_CATEGORIES_MST: 目標カテゴリ（売上向上、スキルアップ等）
- TARGET_TYPES_MST: 目標タイプ（KGI、KPI、OKR等）
- BADGES_MST: バッジ定義
- NOTIFICATION_TEMPLATES_MST: 通知テンプレート
- SYSTEM_SETTINGS_MST: システム設定

トランザクションデータ:
- TARGETS: 目標データ
- PROGRESS_RECORDS: 進捗記録
- WEEKLY_REVIEWS: 週次レビュー
- TARGET_COMMENTS: 目標コメント
- NOTIFICATION_HISTORY: 通知履歴

サマリーデータ:
- DAILY_PROGRESS_SUMMARY: 日次進捗サマリー
- WEEKLY_REVIEW_SUMMARY: 週次レビューサマリー
- MONTHLY_TARGET_PROGRESS_SUMMARY: 月次目標進捗サマリー

ログ:
- OPERATION_LOGS: 操作ログ
- ERROR_LOGS: エラーログ
