============================================================
  目標共有ボード 使い方説明書
============================================================

■ 概要
  会社・部署・チーム・個人の目標を階層的に管理し、
  進捗状況をリアルタイムで共有するためのシステムです。
  ダッシュボードで全体の進捗を一目で把握でき、
  アラートやリマインドで更新漏れを防止します。

■ アクセス方法
  管理者から共有されたURLをブラウザで開いてください。
  PC・タブレット・スマートフォンに対応しています。

============================================================
  初期設定（管理者向け）
============================================================

  以下の順番で初期データを登録してください。

  1. 組織管理
     - 最初に「会社」を登録（例: 株式会社テスト / 種別: 事業部）
     - 次に「部署」を登録（例: 営業部 / 親組織: 株式会社テスト）
     - 必要に応じて「チーム」を登録（例: 営業1チーム / 親組織: 営業部）

  2. メンバー管理
     - 所属メンバーを登録
     - 部署・チームを選択すると自動で名前がセットされます
     - 上長を選択すると自動で上長名がセットされます
     - メールアドレスはアラート・リマインド通知に使用します

  3. 目標管理
     - 会社目標 → 部署目標 → チーム目標 → 個人目標の順に登録
     - 親目標を選択すると階層が紐付きます
     - テンプレートから入力すると、目標種別・カテゴリ等が自動入力されます

  4. 設定（トリガー）
     - 設定ページの「トリガー管理」タブで「トリガー設定」ボタンを押す
     - 毎日8時にアラートチェック、毎週月曜9時にサマリー配信、
       毎月1日9時にレポート生成が自動実行されるようになります

============================================================
  各ページの使い方
============================================================

【ダッシュボード】
  システム全体の状況を一覧表示します。
  - 統計カード: 全目標数、順調数、遅れ数、全体進捗率
  - 進捗分布チャート: 順調/やや遅れ/遅れ/危険/未更新の割合
  - 組織別進捗チャート: 組織ごとの平均進捗率
  - 最近の進捗更新: 直近の更新履歴
  - 直近のアクション: 期限が近いタスク
  - アラート一覧: 検出されたアラート
  - レビュー予定: 終了日が近い目標
  - 達成目標: 本日達成された目標

【目標管理】
  目標の登録・編集・削除を行います。
  - 「新規登録」ボタンで登録フォームを開く
  - 「テンプレートから入力」でテンプレートを適用可能
  - フィルター: 目標レベル（会社/事業部/部署/チーム/個人）で絞り込み
  - 検索: 目標名・責任者名で検索
  - 親目標を設定すると階層ツリーに反映されます

  【入力項目】
    目標名(必須), 目標レベル, 親目標, 目標種別(数値/状態/行動),
    カテゴリ(売上/利益/顧客/業務改善/人材育成/その他),
    目標内容, 達成基準(KPI), 目標値, 単位,
    期間(開始/終了), 責任者, 関係者ID一覧,
    重要度(S/A/B/C), ステータス, 公開範囲

【進捗更新】
  目標の進捗を記録します。
  - 目標を選択し、現在値を入力して保存
  - 進捗率は自動計算されます（現在値 / 目標値 x 100）
  - 進捗状況（順調/やや遅れ/遅れ/危険）も自動判定
  - 親目標がある場合、子目標の平均進捗が親に自動反映されます
  - フィルター: 目標で絞り込み

  【入力項目】
    目標(必須), 更新者, 現在値(必須),
    実施したアクション, 課題・障害, 次のアクション, コメント

【アクション管理】
  目標に紐付くタスク・アクションを管理します。
  - 目標を選択し、具体的なアクション内容を登録
  - 担当者、期限、優先度、ステータスを設定
  - 完了時は貢献度（目標達成への寄与度）を記録
  - フィルター: 目標で絞り込み

  【入力項目】
    目標(必須), アクション内容(必須), 担当者, 期限,
    優先度(高/中/低), ステータス(未着手/進行中/完了),
    貢献度, 備考

【メンバー管理】
  システム利用メンバーを管理します。
  - 部署・チームを選択すると名前が自動セット
  - 上長選択で上長名が自動セット
  - フィルター: 部署で絞り込み

  【入力項目】
    氏名(必須), 部署, チーム, 役職,
    上長, メールアドレス, Slack ID, ステータス

【組織管理】
  会社・部署・チームの組織構造を管理します。
  - 種別: 事業部（会社）、部署、チーム から選択
  - 親組織を設定して階層化
  - フィルター: 種別で絞り込み

  【入力項目】
    組織名(必須), 組織種別, 親組織, リーダー,
    所属人数, ステータス

【テンプレート】
  よく使う目標パターンをテンプレートとして保存します。
  - テンプレートを登録しておくと、目標登録時に
    「テンプレートから入力」で自動入力できます
  - 使用回数は自動カウントされます

  【入力項目】
    テンプレート名(必須), 目標種別, カテゴリ,
    目標文例, KPI例, 推奨期間, 対象レベル, ステータス

【振り返り管理】
  定期的な目標レビューを記録します。
  - レビュー種別: 週次/月次/四半期/期末
  - 対象期間: 年月を選択（例: 2026-02）
  - 進捗率と達成/未達を記録
  - 良かった点・改善点・学び・次期アクションを記載
  - 上長コメント欄あり
  - フィルター: レビュー種別で絞り込み

  【入力項目】
    レビュー種別, 対象期間, 目標(必須), レビュー日,
    進捗率, 達成/未達, 良かった点, 改善点・課題,
    学び・気づき, 次期へのアクション, 上長コメント

【レポート】
  4種類のレポートを表示します。

  - 目標別サマリー: 各目標の進捗率・残り日数・進捗状況を一覧表示
  - 組織別サマリー: 組織ごとの目標数・平均進捗率・達成率を表示
  - 個人別サマリー: メンバーごとの担当目標数・平均進捗・更新頻度を表示
  - 目標ツリー: 会社→事業部→部署→チーム→個人の階層構造と
    各レベルの進捗率を一覧表示

【設定】
  3つのタブがあります。

  [アラート設定]
    進捗遅れや未更新を自動検出する条件を設定します。
    - 条件: 未更新日数 / 進捗遅れ(%) / 期限接近(日)
    - 条件値: 閾値（例: 10 → 進捗率が予定より10%以上遅れで検出）
    - 通知先: 責任者 / 上長 / 全員
    - 通知方法: メール

  [リマインド設定]
    目標の進捗更新を促すリマインドメールを設定します。
    - 対象目標を選択
    - 更新頻度: 毎日 / 週次 / 月次
    - 曜日/日: リマインドする曜日や日付（例: 月,水 や 1,15）
    - 時刻: 06:00〜21:00の1時間刻みで選択

  [トリガー管理]
    自動処理の設定・解除を行います。
    - 「トリガー設定」: 自動チェックを開始
      （毎日8時アラート、毎週月曜9時サマリー、毎月1日9時レポート）
    - 「トリガー削除」: 自動チェックを停止
    - 「手動アラートチェック」: アラートを即時実行して結果を確認
    - アラート検出履歴: 過去に検出されたアラートの一覧

============================================================
  よくある質問
============================================================

Q: 進捗率が自動計算されない
A: 進捗更新ページで「現在値」を入力してください。
   目標値と現在値から自動的に進捗率が算出されます。

Q: 親目標の進捗率が更新されない
A: 子目標の進捗を更新すると、親目標の進捗率は
   子目標の平均値で自動更新されます。

Q: メールが届かない
A: 以下を確認してください。
   1. メンバー管理でメールアドレスが登録されているか
   2. 設定ページでトリガーが設定されているか
   3. アラート設定またはリマインド設定が登録されているか

Q: フィルターで絞り込んだ後、元に戻したい
A: フィルターのドロップダウンで「全て」または「全レベル」を選択してください。

Q: テンプレートはどこから使う？
A: 目標管理ページの「新規登録」を押すと、フォーム上部に
   「テンプレートから入力」の選択欄が表示されます。

Q: データをエクスポートしたい
A: Googleスプレッドシートに全データが保存されています。
   スプレッドシートを直接開いてCSVやExcel形式で
   ダウンロードできます。

============================================================
  スプレッドシートのシート構成
============================================================
  目標マスタ                  … 目標データ
  メンバーマスタ              … メンバーデータ
  部署・チームマスタ          … 組織データ
  進捗データ                  … 進捗更新履歴
  アクション・タスク紐付け    … アクションデータ
  振り返り・レビュー記録      … レビューデータ
  目標テンプレートマスタ      … テンプレートデータ
  アラート設定                … アラート条件設定
  目標更新リマインド設定      … リマインド設定
  未更新・遅延アラートリスト  … アラート検出履歴
  目標別進捗サマリー          … レポート用データ
  組織別進捗サマリー          … レポート用データ
  個人別目標サマリー          … レポート用データ
  階層別目標ツリー            … レポート用データ

============================================================
  注意事項
============================================================
  - スプレッドシートのシート名やヘッダー行は変更しないでください
  - データの直接編集はスプレッドシート上でも可能ですが、
    ID列の値は変更しないでください
  - トリガーは1つのGoogleアカウントにつき1セットです。
    複数人がトリガー設定すると重複実行されます
