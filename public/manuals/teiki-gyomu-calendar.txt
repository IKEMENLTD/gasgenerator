=====================================
定期業務カレンダー自動展開 - 使い方説明書
=====================================

■ このシステムでできること
-------------------------------------
月次・年次・毎週・毎日の定期業務を一元管理し、タスクを自動生成してGoogleカレンダーに展開するシステムです。

- 定期業務のマスタ登録・管理（繰返しルールの設定）
- ルールに基づくタスクの自動生成とカレンダー登録
- タスクの進捗管理（ステータス更新、チェックリスト、コメント）
- 担当者・チーム・カテゴリの管理
- 期限通知・リマインダーの自動メール送信
- ダッシュボードによるリアルタイム状況把握
- 週次・月次レポートの自動集計
- ランキング・バッジによるモチベーション向上
- データのエクスポート（CSV/JSON形式）
- 操作ログ・メール送信ログによる監査対応

■ 利用シーン
-------------------------------------
- 総務: 備品発注、施設点検、契約更新の月次・年次管理
- 人事: 給与計算、社会保険手続き、健康診断手配の定期管理
- 経理: 月次決算、四半期決算、請求書発行の定期管理
- 営業事務: 受注処理、報告書作成の定例タスク管理
- 個人利用: 日常のルーティンワークの抜け漏れ防止


=====================================
■ 画面構成と機能説明
=====================================

1. ダッシュボード
-------------------------------------
アプリ起動時に最初に表示される画面です。

【表示される情報】
- 未完了タスク数: 現在未完了のタスクの総数
- 本日期限: 今日が期限のタスク数
- 本日完了: 今日完了したタスク数
- 期限超過: 期限を過ぎても未完了のタスク数

【グラフ表示】
- カテゴリ別タスク分布: 各カテゴリごとのタスク数を棒グラフで表示
- 優先度別タスク分布: 高・中・低の優先度ごとのタスク数を棒グラフで表示

【直近のタスク】
- 7日以内に期限が来るタスクの一覧を表示
- タスク名をクリックすると詳細画面を表示


2. タスク一覧
-------------------------------------
自動生成されたタスクの一覧を表示・管理する画面です。

【フィルタ機能】
- ステータス: 全て / 未着手 / 進行中 / 完了 / 期限超過 / キャンセル
- カテゴリ: 登録済みカテゴリで絞り込み
- 優先度: 高 / 中 / 低
- 検索: タスク名で部分一致検索

【ソート機能】
- 各カラムのヘッダーをクリックすると昇順/降順で並べ替え

【タスク手動生成ボタン】
- 右上の「タスク手動生成」ボタンで、定期業務マスタに基づくタスクを即座に生成
- 通常はトリガーで自動生成されますが、手動でも実行可能

【タスク詳細画面】
タスク名や「詳細」ボタンをクリックすると詳細モーダルが表示されます。
- タスクの基本情報（ステータス、優先度、担当者、期限日、カテゴリ等）
- チェックリスト: 各項目のチェックボックスでON/OFF切替可能
- ステータス変更: 「進行中」「完了」「キャンセル」ボタンで変更
- 完了処理: 実作業時間の入力とコメント追加が可能
- コメント: タスクに対するコメントの閲覧・投稿


3. 定期業務マスタ
-------------------------------------
繰り返し発生する業務を登録・管理する画面です。

【検索機能】
- 検索バーに文字を入力すると、候補がドロップダウンで表示
- 入力に応じてテーブルもリアルタイムにフィルタリング

【一覧表示】
- ID、業務名、カテゴリ、繰返し種類、担当者、優先度、有効フラグを表示
- 有効フラグ: トグルスイッチで有効/無効を切替可能

【新規登録・編集】
「新規登録」ボタンまたは「編集」ボタンでフォームが表示されます。

入力項目:
- 業務名 (必須): 定期業務の名称
- 説明: 業務の詳細説明
- カテゴリ (必須): ドロップダウンから選択
- 繰返し種類 (必須): 月次 / 年次 / 毎週 / 毎日 / 任意期間
  - 月次: 毎月の何日にタスクを生成するか指定（例: 15日）
  - 年次: 月と日を指定（例: 3月15日）
  - 毎週: 曜日をチェックボックスで選択（複数可）
  - 毎日: 追加設定なし
  - 任意期間: 間隔日数を指定（例: 30日ごと）
- デフォルト担当ユーザー: タスク生成時に自動でアサインされるユーザー
- デフォルト担当チーム: チーム単位でのアサイン
  ※ ユーザーまたはチームの少なくとも一方は必須
- 期限オフセット日数 (必須): タスク生成基準日から期限日までの日数
  - 正の値: 基準日の○日後が期限（例: 7 = 7日後）
  - 0: 基準日当日が期限
  - 負の値: 基準日の○日前が期限
- 通知オフセット日数: 通知を送信する日のオフセット
  - 負の値を指定すると基準日より前に通知（例: -3 = 3日前）
- 通知テンプレート: メール通知に使用するテンプレートを選択
- 優先度: 高 / 中 / 低
- 見積もり時間: 作業にかかる見込み時間（時間単位）
- チェックリスト項目: タスク内のサブ項目を動的に追加
  - 項目名と必須フラグを設定可能
  - 「+ 項目追加」で追加、「x」で削除
- 備考: 自由記述


4. ユーザー管理
-------------------------------------
システムを利用するユーザーを登録・管理する画面です。

【新規登録・編集】
- 氏名 (必須): ユーザーの表示名
- メールアドレス (必須): Googleアカウントのメールアドレス
- チーム: 所属チームをドロップダウンから選択
- ロール: 管理者 / 一般ユーザー / 閲覧者
- GoogleカレンダーID: カレンダー連携用のID（通常はメールアドレスと同じ）
- ステータス (編集時のみ): アクティブ / 停止中
- 備考: 自由記述


5. チーム管理
-------------------------------------
チーム（部署）を登録・管理する画面です。

【新規登録・編集】
- チーム名 (必須): チームの名称
- 説明: チームの役割や概要
- 管理者: チームの管理者ユーザーを選択
- 備考: 自由記述


6. カテゴリ管理
-------------------------------------
業務カテゴリを登録・管理する画面です。
初期状態では「総務」「人事」「経理」「営業」「開発」「その他」が登録されています。

【新規登録・編集】
- カテゴリ名 (必須): カテゴリの名称
- 説明: カテゴリの説明
- 表示順: 一覧での表示順序（数値が小さいほど先に表示）
- 備考: 自由記述


7. 通知テンプレート
-------------------------------------
メール通知の件名・本文のテンプレートを管理する画面です。
初期状態では3つのテンプレートが登録されています:
- タスク作成通知
- 期限リマインダー
- 期限超過通知

【新規登録・編集】
- テンプレート名 (必須): テンプレートの識別名
- 件名 (必須): メールの件名
- 本文 HTML (必須): メール本文（HTML形式）
- 受信者ロール: 担当者 / 上長 / 担当者,上長 / 全員
- 備考: 自由記述

【使用可能な動的変数】
件名・本文内で以下の変数が使用できます:
- {{task_name}} : タスク名
- {{task_id}} : タスクID
- {{due_date}} : 期限日
- {{user_name}} : 担当者名
- {{description}} : タスク説明
- {{priority}} : 優先度
- {{status}} : ステータス


8. バッジ管理
-------------------------------------
ユーザーに付与するバッジ（実績）を管理する画面です。
初期状態では3つのバッジが登録されています:
- 皆勤賞: 3ヶ月連続タスク達成率95%以上
- タスクマスター: 累計50件のタスクを完了
- 期限厳守: 期限超過なしで全タスク完了

【条件設定 (JSON形式)】
- {"type":"continuous_completion","period":"monthly","count":3}
  → 3ヶ月連続で達成率95%以上
- {"type":"total_completed","count":50}
  → 累計50件のタスク完了
- {"type":"no_overdue","period":"monthly"}
  → 期限超過なしで全タスク完了
- {"type":"category_completed","category_id":"CAT001","count":10}
  → 特定カテゴリで10件完了


9. レポート
-------------------------------------
業務の達成状況を集計したレポートを確認する画面です。

【週次レポートタブ】
- 毎週月曜日に自動集計される前週のレポート
- 全体およびユーザー別の総タスク数、完了数、超過数、達成率を表示

【月次レポートタブ】
- 毎月1日に自動集計される前月のレポート
- 全体およびユーザー別の統計を表示


10. ランキング
-------------------------------------
ユーザーのタスク達成状況をランキング形式で表示する画面です。

- 順位: スコアに基づく順位
- スコア: 完了タスク数 x 10 - 期限超過数 x 5 で計算
- 完了数: 累計の完了タスク数
- 超過数: 累計の期限超過タスク数
- 合計時間: 完了タスクの合計作業時間


11. システム設定
-------------------------------------
システム全体の設定を管理する画面です。

【設定項目】
- DEFAULT_NOTIFICATION_TIME: デフォルト通知時刻
- CALENDAR_SYNC_ENABLED: Googleカレンダー連携の有効/無効
- DATA_RETENTION_DAYS: データ保持期間（日数）
- MAX_EXPORT_ROWS: エクスポート最大行数

【システム操作】
- シート初期化: 全シートのヘッダー作成と初期データ投入
  ※ 既存データは削除されません。初回利用時に実行してください。
- トリガー設定: 自動実行トリガー（8種類）を一括設定


12. 操作ログ
-------------------------------------
システムの操作履歴を確認する画面です。

【操作ログタブ】
- 日時、ユーザー、操作内容、対象エンティティ、状態（成功/失敗）を表示
- 最新100件を表示

【メール送信ログタブ】
- 日時、宛先、件名、関連タスク、送信状態を表示
- 通知メールの送信履歴を確認可能


13. データエクスポート
-------------------------------------
各種データをCSVまたはJSON形式でダウンロードする画面です。

【エクスポート対象】
- 定期業務マスタ
- 生成タスク履歴
- 週次レポート
- 月次レポート
- 操作ログ
- メール送信ログ

【操作手順】
1. エクスポート対象をドロップダウンから選択
2. 必要に応じて開始日・終了日で期間を絞り込み
3. 形式（CSV / JSON）を選択
4. 「エクスポート実行」ボタンをクリック
5. ダウンロードリンクが表示されるのでクリックしてダウンロード


=====================================
■ 自動実行トリガー
=====================================

以下のトリガーが自動的に実行されます（システム設定画面の「トリガー設定」ボタンで設定）。

1. 月次タスク自動生成（毎月1日 午前2時）
   - 月次および該当月の年次定期業務からタスクを自動生成
   - Googleカレンダーにイベント登録

2. 年次タスク自動生成（毎年1月1日 午前3時）
   - 年次定期業務からタスクを自動生成

3. 事前通知メール送信（毎日 午前8時）
   - 通知予定日が当日のタスクの担当者にメール送信

4. 期限超過タスク通知（毎日 午前9時）
   - 期限超過タスクの担当者と上長にメール送信
   - ステータスを自動的に「期限超過」に更新

5. 週次レポート集計（毎週月曜 午前1時）
   - 前週のタスク達成状況を自動集計

6. 月次レポート集計（毎月1日 午前4時）
   - 前月のタスク達成状況を自動集計

7. ダッシュボードデータ更新（毎日 午前0時）
   - ダッシュボードの集計データとランキングを更新

8. 達成バッジ自動付与（毎日 午前1時）
   - バッジの付与条件を評価し、該当ユーザーにバッジを付与


=====================================
■ 初回利用時の手順
=====================================

1. アプリのURLにアクセスする
2. Googleの承認画面が表示されたら「許可」をクリック
3. 左メニューの「システム設定」を開く
4. 「シート初期化」ボタンをクリック
   → 全シートのヘッダーと初期データが作成されます
5. 「トリガー設定」ボタンをクリック
   → 自動実行トリガーが設定されます
6. 「ユーザー管理」でユーザーを登録
7. 必要に応じて「チーム管理」「カテゴリ管理」でマスタを整備
8. 「定期業務マスタ」で繰返し業務を登録
9. タスクは次回トリガー実行時に自動生成されます
   すぐにテストしたい場合は「タスク一覧」の「タスク手動生成」を使用


=====================================
■ 操作のコツ
=====================================

- モバイル端末ではヘッダー左のメニューアイコンからナビゲーションを開閉できます
- テーブルのカラムヘッダーをクリックするとソートできます
- 検索バーは入力に応じてリアルタイムに絞り込まれます
- タスクの完了時に実作業時間を入力すると、レポートやランキングに反映されます
- チェックリストの必須項目を全て完了してからタスクを完了にしてください
- 定期業務マスタのトグルスイッチでタスク生成の有効/無効を切り替えられます
- エクスポート機能で定期的にデータのバックアップを取ることを推奨します
